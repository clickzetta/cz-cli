## 功能

本命令用于展示指定 schema 的详细信息，包括其结构、字段、索引等，帮助用户更好地了解和管理数据。

## 语法

```SQL
DESC[RIBE] SCHEMA [EXTENDED] schema_name;
```

## 参数说明

1. **DESC[RIBE]**：DESC 和 DESCRIBE 可以互换使用，都表示查询 schema 信息的命令。
2. **schema_name**：指定需要查询的 schema 名称。
3. **EXTENDED**（可选）：加入此关键字后，将展示更多关于 schema 的扩展信息，例如字段的默认值、索引类型等。

## 使用示例

1. 查询名为 `ods_schema` 的 schema 的基本信息：

```
DESCRIBE SCHEMA tpch100g;
+--------------------+-------------------------+
|     info_name      |       info_value        |
+--------------------+-------------------------+
| name               | tpch100g                |
| creator            | UAT_TEST                |
| created_time       | 2024-11-29 21:10:11.892 |
| last_modified_time | 2024-11-29 21:10:11.892 |
| comment            |                         |
+--------------------+-------------------------+
```

2. 查询名为 `ods_schema` 的 schema 的详细信息（包括默认值、索引等）：

```
DESCRIBE SCHEMA EXTENDED tpch100g;
+--------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
|     info_name      |                                                                                                                                                                                     |
+--------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| name               | tpch100g                                                                                                                                                                            |
| creator            | UAT_TEST                                                                                                                                                                            |
| created_time       | 2024-11-29 21:10:11.892                                                                                                                                                             |
| last_modified_time | 2024-11-29 21:10:11.892                                                                                                                                                             |
| comment            |                                                                                                                                                                                     |
| properties         | (("type","hive"),("hive_schema","tpch100g"),("hive_metastore_uris","192.168.20.xxx:9083"),("fs.defaultfs","hdfs://xxxx:9000")                                                       |
| type               | external                                                                                                                                                                            |
+--------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
```

3. 查询名为 `dws_schema` 的 schema，并使用 DESC 关键字：

```
DESC SCHEMA tpch100g;
+--------------------+-------------------------+
|     info_name      |       info_value        |
+--------------------+-------------------------+
| name               | tpch100g                |
| creator            | UAT_TEST                |
| created_time       | 2024-11-29 21:10:11.892 |
| last_modified_time | 2024-11-29 21:10:11.892 |
| comment            |                         |
+--------------------+-------------------------+
```
