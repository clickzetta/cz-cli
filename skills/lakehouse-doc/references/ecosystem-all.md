## 云器Lakehouse生态工具连接

云器 Lakehouse 作为一款功能强大的数据平台，支持与众多生态工具无缝连接，以便您能够更加便捷地管理和分析数据。支持的生态工具包括数据库客户端工具、BI 及可视化工具、ETL 工具等。当前支持的生态工具如下：

| 分类 | 名称 |
| :--- | :--- |
| 客户端 | [SQL Workbench/J](eco_integration/sqlworkbench-j-lakehouse.md) |
| 客户端 | [DBeaver](eco_integration/dbeaver-lakehouse.md) |
| 客户端 | [DataGrip](eco_integration/datagrip-lakehouse.md) |
| BI及可视化 | [Metabase](metabase.md) |
| BI及可视化 | [Tableau   ](<TableauConnectToLakehouse.md>) |
| BI及可视化 | [Superset](eco_integration/superset.md) |
| BI及可视化 | [rath](eco_integration/rath.md) |
| BI及可视化 | [Streamlit](eco_integration/streamlit.md) |
| BI及可视化 | [Zeppelin](eco_integration/Zeppelin.md) |
| BI及可视化 | [帆软 ](<FineBI.md>) |
| ETL | [DataX](eco_integration/datax.md) |
| ETL | [DBT](eco_integration/dbt.md) |
| ETL | [Airbyte](airbyte.md) |
| 开放引擎 | [Trino](eco_integration/trino.md) |
| 开放引擎 | [Spark](spark-connector-summary.md) |
| 其他 | [MindsDB](mindsdb.md) |

对于列表之外的生态工具，您可以根据客户端支持的连接方式，考虑使用 JDBC 驱动或 SQLAlchemy 创建自定义连接。
