## 功能

`SHOW USERS` 命令用于查询当前数据库空间下的所有用户信息。通过该命令，用户可以快速了解当前空间中的用户列表，便于进行用户管理和权限控制。

## 语法

```SQL
SHOW USERS [LIKE pattern];
```

**参数说明**

- `LIKE pattern`（可选）: 通过对象名称过滤命令输出。该过滤器使用不区分大小写的模式匹配，支持 SQL 通配符（`%` 和 `_`）。例如，`LIKE '%test%'` 将只显示包含 "test" 文本的用户名称。

## 使用示例

### 示例 1: 查询当前空间下的所有用户

```SQL
SHOW USERS;
```

执行该命令后，将列出当前空间下的所有用户信息。

### 示例 2: 根据用户名关键字查询用户

```SQL
SHOW USERS LIKE '%admin%';
```

执行该命令后，将列出当前空间下用户名中包含 "admin" 的所有用户信息。

### 示例 3: 查询当前空间下特定前缀的用户

```SQL
SHOW USERS LIKE 'user_%';
```

执行该命令后，将列出当前空间下用户名以 "user_" 为前缀的所有用户信息。

## 注意事项

- 使用 `LIKE` 子句时，请注意避免使用过多的通配符，以免影响查询性能。
- 请确保在执行命令时，当前用户具有足够的权限来查询其他用户信息。
