# 功能描述

描述指定对象的详细信息。DESCRIBE 可以缩写为 DESC。

# 语法

```
DESC[RIBE] <object_type> [EXTENDED] <object_name>
```

* `EXTENDED` 可选，使用 `EXTENDED` 可以展示更多信息。
