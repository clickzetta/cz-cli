## 功能
查看作业的详细信息，包括作业的配置、运行状态、输出结果等。

## 语法
```
DESC JOB 'jobid';
```

## 参数说明

- `jobid`：作业运行的唯一标识符，字符串格式。在作业运行过程中，日志会显示该标识符，同时在 `show jobs` 命令的输出结果中也会展示该标识符。

## 使用场景

1. 当您需要查看某个作业的配置信息和运行状态时，可以使用 `DESC JOB` 命令。
2. 如果您需要了解作业的输出结果，以便进行后续分析和处理，也可以通过该命令获取相关信息。

## 示例

1. 查看作业ID为'201xxxxxxxxxx'的详细信息：
```
DESC JOB '201xxxxxxxxxx'
```

2. 假设您在 `show jobs` 命令的输出结果中发现一个名为 'example_job' 的作业，您可以通过以下命令查看该作业的详细信息：
```
DESC JOB 'example_job'
```


通过以上示例，您可以灵活地使用 `DESC JOB` 命令来查看和管理您的作业。
