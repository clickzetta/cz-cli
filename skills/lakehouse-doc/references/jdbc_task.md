# JDBC任务

JDBC脚本是数据开发中的一类重要任务节点，通过它可以编写SQL代码来连接支持JDBC协议的数据源，实现数据的增、删、改、查等操作。本文档将为您提供详细的操作指南和使用示例，帮助您更高效地使用JDBC脚本。

## 操作指南

1. **新建任务**：在数据开发界面中，点击新建任务按钮，进入任务配置页面。

2. **选择任务类型**：在任务配置页面，选择“JDBC脚本”作为任务类型。

   ![](.topwrite/assets/20240618-142805.jpg)

3. **配置数据源**：在任务配置页面，选择一个已有的数据源或新建一个数据源。选择数据源后，还需选择该数据源下的具体数据库。

   ![选择数据源和数据库](.topwrite/assets/image_1701175218238.png =640)

4. **编写SQL脚本**：在SQL编辑区域，编写您需要执行的SQL语句。您可以编写多条SQL语句，它们将按照编写顺序依次执行。

5. **运行任务**：点击“运行”按钮，执行您编写的SQL脚本。执行结果将在下方的结果显示区域展示。

   ![运行JDBC任务](.topwrite/assets/image_1701175231864.png)

6. **任务调度**：和SQL任务一样，JDBC任务可以直接实现周期调度和运维，并通过设置任务依赖实现与其他任务的工作流编排。

## 支持的数据源及版本

目前，JDBC任务支持以下数据源及其对应版本：

| 数据源        | 版本号    |
| ------------ | -------- |
| MySQL        | 5.1.49   |
| Hive         | 3.1.2    |
| ClickHouse   | 0.26     |
