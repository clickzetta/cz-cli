# 导出数据概览

Lakehouse大数据计算引擎提供了灵活的数据导出功能，允许用户将数据导出为CSV文件格式，便于在本地或其他数据分析平台中使用同时也支持使用数据集成工具，数据集成工具支持了丰富的数据源比如MySQL、PostgreSQL、HIVE、TIDB、StarRocks
