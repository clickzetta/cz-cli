## 功能描述
本命令用于展示当前工作空间（workspace）内所有可用的函数，包括内置函数和外部函数。通过该命令，用户可以快速了解并选择所需的函数进行数据查询和处理。

## 语法格式
```SQL
SHOW FUNCTIONS [ LIKE 'pattern' ]
```

## 参数说明
- **LIKE 'pattern'**：该参数用于根据函数名称进行过滤，只展示与指定模式匹配的函数。支持使用 SQL 通配符，包括百分号（%）表示任意数量的字符，下划线（_）表示任意单个字符。

## 使用示例

1. 查看所有函数：
   ```SQL
   SHOW FUNCTIONS;
   ```

2. 根据函数名称部分关键词过滤结果，例如查找包含 "ABS" 的所有函数：
   ```SQL
   SHOW FUNCTIONS LIKE '%ABS%';
   ```

3. 查找特定的函数，例如查找名为 "SQRT" 的函数：
   ```SQL
   SHOW FUNCTIONS LIKE 'SQRT';
   ```

4. 使用通配符查找以 "DATE" 开头的所有函数：
   ```SQL
   SHOW FUNCTIONS LIKE 'DATE%';
   ```

5. 组合使用通配符进行更精确的查找，例如查找包含 "DATE" 并且以 "DIFF" 结尾的函数：
   ```SQL
   SHOW FUNCTIONS LIKE 'DATE%DIFF';
   ```

## 注意事项
- 当不指定 LIKE 子句时，将展示所有可用的函数。
- 使用通配符时，请确保它们紧跟在 LIKE 关键字之后，且前面不要有空格。
- 在使用 SQL 通配符时，请注意避免使用过多的通配符，以免影响查询性能。

通过以上示例和说明，用户可以更加方便地查找和使用所需的函数，从而提高数据处理和分析的效率。
