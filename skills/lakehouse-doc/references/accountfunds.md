# 账户资金

## 简介

账户资金是租户在云器平台上用于结算使用产品服务后相应费用的账户，所有 Region 都统一使用此账户进行结算。同时，您也可以在本功能中查看资金的收支明细。

## 功能权限

对于有多用户的租户，账户资金功能仅限具有“账户管理员”权限的用户访问。

## 分类

账户资金目前分为现金账户、代金券账户两类。

* 现金账户：可以通过对公转账、微信、支付宝线上充值的方式预充值资金
* 代金券账户：云器为租户发放的代金券。

## 功能路径

登录 Lakehouse Studio -> 费用中心 -> 账户资金，如下图：
![](.topwrite/assets/image_1748336067204.png)
* 现金金额：租户当前可用余额。通过对公转账、微信、支付宝等方式充值的金额，会根据消费情况实时扣除。
* 未结清金额：租户当前存在的未结清金额，当现金余额为0且无可用代金券时，继续使用服务就会产生未结清金额，存在未结清金额时，需要及时完成充值，以免影响服务的正常运行。
* 代金券：租户当前可用代金券数量。

### 现金

#### 充值汇款

支持微信、支付宝在线充值及线下对公转账。不同充值方式的差异如下：

| 充值方式 | 说明 | 到账时间 |
| :--- | :--- | :--- |
| 支付宝充值 | 通过支付宝账户进行充值，可以使用支付宝余额、账户绑定的储蓄卡、信用卡、亲情卡等方式充值。                                                   | 实时到账    |
| 微信充值  | 通过微信账户充值，可以使用微信零钱余额、绑定的储蓄卡，不支持通过信用卡充值。                                                         | 实时到账    |
| 对公转账  | 需要使用开户名与云器租户实名认证主体一致的银行账户进行汇款；可选择通过网银、手机银行客户端、银行柜台等方式将款项汇款至云器的对公账号中；汇款后需联系您对应的商务，将款项充值至您的云器现金账户中。 | 3个工作日 |

充值汇款操作指引：

在现金账户页面点击“充值汇款”按钮，进入充值汇款页面，如下图：
![](.topwrite/assets/image_1748336104481.png)
如选择线上充值，首次充值需同意《云器科技用户协议》，点击微信或支付宝选择充值方式，输入充值金额，点击“立即充值”，即会生成付款二维码。使用手机端对应APP扫描二维码并完成支付，如下图：
![](.topwrite/assets/image_1747637006371.png)
如选择对公转账，可进入转账汇款页面，查看云器收款信息，按照收款信息将需充值金额转账至云器账户，并联系商务完成收款。如下图：
![](.topwrite/assets/image_1747637124644.png)
完成充值后，可回到现金账户页面，查看对应充值的交易明细。当存在欠费时，充值到账后会实时抵扣欠费金额。

异常情况：若选择线上充值，但超时未完成支付，将提示充值失败，如下图。此时需要您刷新页面后重新发起充值。
![](.topwrite/assets/充值失败.jpeg)

**关于发票**：如您需要开具发票，可联系商务获取。发票将根据您在云器平台的消费记录开具，而非根据实际充值金额，请知晓。

**关于退款**：暂不支持线上发起退款。如因账户注销等情况需要发起退款，请联系商务线下完成退款，退款仅支持原路返回支付账户。

#### 交易明细

交易明细展示了现金账户的收支明细，您可以通过此功能查看账户余额的变更记录，如下图：
![](.topwrite/assets/image_1747637294249.png)
* 交易流水：平台交易的唯一凭证
* 交易时间：交易发生时间
* 类型：包含消费、充值、退款三种类型
* 账户：当前仅有现金账户
* 交易金额：交易发生时产生的金额
* 可用余额：交易发生后剩余可用现金金额
* 渠道：交易发生渠道，针对充值类型交易，记录充值来源渠道，例如：对公转账、微信、支付宝
* 交易查询：可通过设定交易时间、交易类型等条件，查询特定场景下的交易记录。

### 代金券

在账户资金页面顶部点击“代金券”，进入代金券页面。默认展示租户当前可用代金券，可切换至“全部”、“不可用”查看对应状态的代金券，如下图：
![](.topwrite/assets/image_1747637516784.png)
* 可用代金券：代金券在生效期内，且有可用余额
* 不可用代金券：代金券未到达可用期、代金券已过可用期、代金券在有效期内但余额已用尽，以上情况的代金券均为不可用。其中，未到达可用期的代金券在到达可用期后会自动置为可用状态；已过期的代金券不可恢复。
**注**：所有代金券均由云器平台发放，不可转让、提现。

#### 代金券详情

点击“查看明细”按钮可查看代金券消费明细，如下图：
![](.topwrite/assets/image_1747637759011.png)
![](.topwrite/assets/image_1747637740486.png)
* 代金券状态：生效中、未生效、已过期
* 代金券面额：初始发放代金券的总金额
* 余额：代金券剩余金额
* 使用时间：代金券有效期，在有效期内均可用于抵扣账单
* 地域范围：代金券支持抵扣的实例归属的 Region 范围
* 产品范围：代金券支持抵扣的计费项范围
* 交易明细：涵盖代金券消费记录，以计费项维度展示抵扣记录，支持通过交易时间筛选
