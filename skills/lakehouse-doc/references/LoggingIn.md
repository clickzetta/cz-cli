# 开通使用

本指南将帮助您完成云器账号注册、开通服务实例和实例初始化操作，助您顺利开启云器之旅。

## 账号注册

在使用云器服务前，您需要注册一个云器账户。

1. **访问账户注册地址**：[accounts.clickzetta.com](https://accounts.clickzetta.com/register)，点击“注册”按钮，在注册页面按照提示填写信息。

:-: ![](.topwrite/assets/image_1736314854109.png =734)

2. **填写手机号**：在云器注册页面，请输入一个可以正常接收短信的手机号，然后点击“获取验证码”按钮。系统将发送一条验证码短信至您的手机，请将收到的4位验证码填入“验证码”输入框，并点击“下一步”。![](.topwrite/assets/image_1739260609977.png =762)

3. **创建用户**：在此步骤中，您需要为您的账户创建第一个用户。请填写“用户名”、“密码”和“确认密码”。用户名长度需在5-32个字符之间，支持英文、数字和下划线，且需以字母开头。密码长度需在8-32个字符之间，必须包含大写字母、小写字母、数字和特殊符号。

4. **完成注册**：点击“完成”按钮后，您将看到注册成功提示，并收到账户登录地址。请妥善保存此地址，以便日后登录。注册时创建的用户将自动成为该账户的管理员。点击提示窗口的“去登录”按钮，使用注册时的用户名和密码登录您的账户。

## 登录账户

注册账户后，您可使用账户及您在此账户下的用户名、密码进行登录。

1. **使用账户登录地址**：使用您的<账户名>.accounts.clickzetta.com URL进行登录。在登录提示框中输入正确的用户名和密码，点击“登录”按钮。如忘记用户名或密码，请点击“忘记用户名？”或“忘记密码？”按钮，使用注册时的手机号进行验证后找回。

:-: ![](.topwrite/assets/image_1736314906745.png =776)

2. **使用通用登录地址**：访问[accounts.clickzetta.com](https://accounts.clickzetta.com/login)进行登录。输入您的8位账户名（完成注册时显示），点击“登录”后，进入账户的登录页面。在登录提示框中输入正确的用户名和密码，点击“登录”按钮。如忘记账户名，请在账户登录页面点击“找回账户”，输入注册时的手机号进行验证后找回。

## 创建Lakehouse服务实例

登录您的云器账户后，您将进入管理中心首页。您可以在管理中心开通Lakehouse服务实例。请按照以下步骤操作：

* 在管理中心“首页”中找到Lakehouse产品卡片，点击“免费开通”按钮。![](.topwrite/assets/20250211-191828.jpeg)
* 在弹出窗口中选择云服务商和开通地域，点击“确认”按钮，完成Lakehouse服务实例的创建。目前云器Lakehouse支持在阿里云上海、腾讯云上海和亚马逊云科技（中国）北京三个区域开通Lakehouse服务实例。

:-: ![](.topwrite/assets/create_instance.png =616)

* 请注意，目前限制每个账户只能创建一个Lakehouse服务实例。

## 实例初始化

创建Lakehouse服务实例后，您可以在管理中心的“我的订阅”中找到并进入该服务实例。系统将自动进行初始化，包括：

1. 创建第一个名为“quick\_start”的工作空间。
2. 创建“quick\_start”工作空间内的第一个虚拟计算集群（Virtual Cluster），默认处于停止状态。
3. 创建系统服务用户`sysservice_clickzetta`和`sysservice_auto_mv`。
4. 将创建Lakehouse服务实例的用户设置为服务实例的管理员角色以及工作空间“quick\_start”的空间管理员（workspace\_admin）角色。此时，您将具备Lakehouse服务实例及“quick_start”工作空间的所有操作权限。后续，您可以通过用户和权限管理功能添加更多用户并分配相应权限。

现在，您已经完成了使用云器Lakehouse的准备工作。您可以查看其它快速开始文档，了解如何快速上手使用Lakehouse。如需更详细的操作指导，请参考“[操作指南](studio_manual.md)”章节。

## 开通 DataGPT 服务

DataGPT 服务现已在阿里云华东2（上海）区域上线。系统将与同区域（华东2（上海））的阿里云 Lakehouse 实例实现默认对接。对于未在阿里云华东2（上海）开通的 Lakehouse 实例，您可以直接在 DataGPT 的数据源页面中手动添加，无需进行勾选操作。

* 在管理中心“首页”中找到 DataGPT 产品卡片，点击“免费开通”按钮。

&#x20;      ![](.topwrite/assets/20250211-192212.jpeg =731)

* 弹出窗口中会默认指定**云服务商**阿里云和**地域**华东2（上海）。系统提供了“**同时开通阿里云-华东2（上海）的 Lakehouse 实例作为默认数据源**”选项：
  * **勾选（推荐新用户）**：将自动开通阿里云-华东2（上海）的 Lakehouse 实例作为默认数据源，无需手动配置。
  * **不勾选**：系统不会自动开通华东2（上海）地区的 Lakehouse 实例，您可以在服务开通后，在数据源管理页面手动添加。请注意，此情况下 **DataGPT 不会包含预置的样例数据**。

&#x20;      ![](.topwrite/assets/20250211-192304.jpeg =618)

* 点击“开通”，稍等片刻后即可进入使用界面。

&#x20;      ![](.topwrite/assets/20250211-193618.jpeg =787)
