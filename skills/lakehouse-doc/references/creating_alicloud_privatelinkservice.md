# 创建阿里云终端节点服务

当需要使用云器 Lakehouse 访问您在云厂商 VPC 内的服务（如自建的 MySQL 数据库）时，您首先需要在阿里云创建终端节点服务，并将需要被访问的服务配置为该终端节点服务的服务资源。

:-: ![](.topwrite/assets/private_link访问客户-例子.png)

## 操作步骤

以下操作需要在阿里云控制台环境下完成。

1\. 请参考https\://help.aliyun.com/zh/privatelink/user-guide/create-and-manage-endpoint-services? 完成前提条件，包括“开通私网连接服务”、“创建负载均衡实例”。

2\. 在阿里云控制台左侧导航栏单击私有连接 - 终端节点服务 ，选择与当前Lakehouse服务实例相同的地域创建终端节点服务。您可以在 Lakehouse 服务的“创建终端节点服务”弹窗上方，找到当前服务实例所在的地域和可用区。请按照当前服务实例的地域和可用区创建终端节点服务，否则会导致 Lakehouse 无法创建对应的终端节点，从而无法建立 PrivateLink 连接。

:-: ![](.topwrite/assets/image_1733063729452.png =381)

3\. 请根据您在阿里云上服务的实际情况配置终端节点服务创建的其他选项。请注意，“服务付费者”选项请选择“服务提供方”，否则会影响正常创建连接。

:-: ![](.topwrite/assets/image_1733063748752.png =636)

4\. 在终端节点服务创建完成后，请点击终端节点的名称，进入终端节点服务详情页面，添加 Lakehouse 的云账号 ID 为白名单。

:-: ![](.topwrite/assets/image_1733063764046.png =735)

^

:-: ![](.topwrite/assets/image_1733063781421.png =367)

5\. 完成上述操作后，请回到Lakehouse页面。在“创建终端节点服务”弹窗中，填写上述步骤中创建的终端节点服务实例 ID（在阿里云终端节点服务详情页面中可找到）和名称，并点击 “确定” 。

:-: ![](.topwrite/assets/image_1733063795296.png =442)
