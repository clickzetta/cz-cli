# **计费账单**

## 简介

计费账单功能展示了您在云器Lakehouse平台的消费情况，账单会根据产品的使用情况持续更新。

## 功能权限

对于有多用户的租户，计费账单功能仅对管理员用户开放。

## 账单更新

* 计算、网络类资源账单的更新周期为每小时一次。
* 存储类资源账单的更新周期为每天。系统会在一天内多次采集数据，并于次日凌晨更新前一天的账单实际用量数据。

## 数据范围

您可以通过计费账单功能查看自2023年5月起产生的账单数据。

## 账单总览

通过账单总览可了解使用云器Lakehouse的总成本，也可按照计算资源、存储、网络分别查看对应分类下的成本。

您可通过指定可用区、实例、时间区间、工作空间等条件来查询和追溯成本来源，了解不同工作空间的成本分布。

## 账单明细

账单明细展示每日账单情况，可查看资源原价、折扣等信息，并支持明细导出。
![](.topwrite/assets/image-20250227112939722.png)

| 字段 | 说明 |
| :--- | :--- |
| 消费时间 | 账单对应的消费日期，以天为单位展示账单数据。 |
| 计算用量 | 计算类型计费项产生的用量汇总，包括：数据集成、任务调度、通用型计算集群、分析型计算集群、Serverless作业、流式集成。 |
| 存储用量 | 存储计费项产生的用量。 |
| 网络用量 | 网络计费项产生的用量。 |
| 原价 | 各计费项单价 × 用量后汇总金额 |
| 折后消费 | 各计费项单价 × 用量 × 计费项折扣后汇总金额 |
| 折扣 | 此处折扣为综合折扣，而非每个计费项的实际折扣。计算规则为：折扣 = 折后消费 / 原价 × 100%。 |

具体产品的计费项定价和计费规则，请参考[定价和计费](pricing.md)。

**账单明细导出**：点击列表右上角的“导出明细”按钮，即可导出计费账单明细。导出文件格式为 XLSX。

## 计费总览

计费总览展示了账户当前余额、今日消费金额、昨日消费金额、本月累计消费金额的统计信息，方便您快速了解消费趋势，管理账户资金。

![](.topwrite/assets/image-20250227113055302.png =300)

* 余额：账户当前剩余可用金额，充值、消费等操作均会导致余额变更。
* 今日消费：当日账单累计消费金额，每小时更新。
* 昨日消费：昨日账单汇总金额。
* 本月累计消费：当月账单累计消费金额（含当日消费），每小时更新。

## 账单历史

账单历史提供全部账单记录的汇总，默认展示当年按月汇总的数据，可通过切换年份查看历史年份的账单记录。
