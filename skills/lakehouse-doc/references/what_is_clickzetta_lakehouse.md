# 云器 Lakehouse 产品简介

云器 Lakehouse 是一种创新的湖仓架构服务，通过全面托管的湖仓一体架构，彻底革新了企业数据管理方式。云器 Lakehouse 平台并非传统数据库技术或“大数据”软件平台的简单延伸，而是基于全新的云原生设计理念从零打造，并配备了一个全新设计的 SQL 计算引擎。这一架构赋予云器 Lakehouse 企业级数据湖/数据仓库的所有能力，同时还拥有众多特殊功能和独特优势。

通过无缝整合数据仓库、数据湖探索、实时与批量数据处理以及商业智能报表等关键组件，云器 Lakehouse 能够满足企业在不同发展阶段对动态扩展性、成本控制和性能优化的需求。我们专注于存储与计算分离以及 Serverless 弹性计算，同时结合 AI 优化工具，极大简化了数据开发与管理流程。这不仅帮助客户更快、更高效地获取洞察，还以无与伦比的灵活性和易用性充分释放数据的全部潜力。

云器 Lakehouse 采用存算分离和 Serverless 弹性算力技术，并具备智能优化特性，旨在简化数据开发和管理工作，加速企业数据洞察和价值实现。

## 产品主要特性

### 真正实现全托管的数据平台云服务

云器 Lakehouse 是一项真正的全托管云服务，这意味着：

* 您无需选择、安装、配置或管理任何硬件，无论是虚拟还是物理设备，全都无需操心。
* 实际上，您也无需进行任何软件的安装、配置或管理操作，一切交给我们来完成。
* 所有的日常维护、管理、升级和性能调优，均由云器 Lakehouse 的专业团队为您精心处理。
* 云器 Lakehouse 完全运行在云端基础设施之上。我们服务的每个组件（可选的命令行客户端、驱动程序和连接器除外）都托管在公有云环境中。
* 在计算需求方面，云器 Lakehouse 使用虚拟计算实例；而对于持久化数据存储，则依托专用的存储服务。
* 与传统的软件包产品不同，云器 Lakehouse 并非需要您自行安装的软件。我们负责所有软件的安装和更新工作，确保为用户提供无缝且无忧的使用体验。

### 下一代SQL数据处理引擎

云器 Lakehouse 搭载的 SQL 数据处理引擎能够同时满足批量处理、实时计算和在线分析的需求。无需依赖多个组件组合，用户可以通过单一引擎高效地进行数据处理和分析，显著降低数据应用开发的复杂性，加快价值的实现。

该引擎基于云原生架构，采用存算分离模式和向量化执行引擎，确保在同等资源规模下的性能领先，在 TPCDS 10TB/TPCH 100GB/SSB 100GB 测试集上领先业界主流产品。

### Serverless弹性架构

云器 Lakehouse 采用存算分离的虚拟计算集群设计，为 ETL 和分析场景提供专门的执行引擎，实现计算优化。用户可以根据业务需求创建不同的虚拟集群，实现业务间的相互隔离，并满足对高吞吐和高性能的不同需求。

平台能够智能感知业务负载，自动管理集群的启停和弹性并发，实现低成本和高服务水平协议（SLA）的双重目标。

:-: ![](.topwrite/assets/image_1691805470858.png =712)

### 开放、无数据锁定

云器 Lakehouse 的存储层兼容主流的开放存储格式，元数据服务以 Catalog 形式提供，允许外部引擎在通过身份认证和鉴权后访问数据文件。在统一的安全策略框架下，平台提供最佳的开放性，并借助丰富的开源生态系统，帮助企业释放更大的数据价值。

:-: ![](.topwrite/assets/image_1691805492197.png =664)

### 统一数据资产

云器 Lakehouse 支持结构化数据的 Table 形式存储和非结构化数据的文件及目录形式存储。平台提供基于访问控制列表（ACL）的精细化权限控制，确保数据资产的统一管理和安全访问。此外，平台还支持非结构化数据的探索与分析，进一步提升数据利用效率。

:-: ![](.topwrite/assets/image_1691805420547.png =640)

### 数据开发与管理的一体化解决方案

云器 Lakehouse 将数据处理引擎的能力与平台工具无缝集成，为用户提供统一的 Web 使用体验。平台覆盖数据集成、开发运维、数据目录、安全治理、数据分析和系统管理等多个方面，通过存算分离、按需弹性虚拟集群以及自动化运维和优化等特性，简化了基础设施的运维管理工作，为用户提供了 SaaS 化的一体化数据平台解决方案。
