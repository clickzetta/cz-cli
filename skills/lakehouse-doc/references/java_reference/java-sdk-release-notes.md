# Java SDK Release Notes
2023.10.20

本发行说明包含以下关于 Java SDK 的更新和改动：

- 行为变更
- 新增特性
- 缺陷修复

## 版本 1.1.1
### 缺陷修复
- 修复了 igs 在特定场景下的正确性问题，提高了程序的稳定性和可靠性。
- 修复了 JDBC Database Metadata 的 getColumns 接口中 boolean 类型数据展示不准确的问题。

### 示例
如果您在使用 igs 时遇到正确性问题，升级到 1.1.1 版本即可解决。同样，如果您在使用 getColumns 接口时遇到 boolean 类型数据展示不准确的问题，也可以通过升级到此版本获得修复。

## 版本 1.1.2
### 缺陷修复
- 修复了 set query_tag 功能，现在支持过滤 value 中多余的引号，提高了查询的准确性。
- 优化了 put/get volume 功能，现在支持展示上传和下载的结果，方便用户查看操作进度。

### 示例
1.  当您需要设置 query_tag 时，可能会遇到 value 中包含多余引号的问题。升级到 1.1.2 版本后，系统可以正确过滤这些引号，确保查询的准确性。
2.  在使用 volume 上传和下载文件时，您可能需要查看操作进度。升级到 1.1.2 版本后，您可以方便地查看上传和下载的结果。

## 版本 1.1.3（待实施）
### 缺陷修复
- 修复了 igs client 在特殊网络故障场景下无法正常退出的问题，提高了程序的稳定性。

### 预期示例
如果您遇到 igs client 在特殊网络故障情况下无法退出的问题，该问题将在 1.1.3 版本中得到解决。这将提高程序的稳定性，确保在遇到类似问题时能够正常退出。
