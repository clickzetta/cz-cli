# 客户端、Connector、SDK下载

本页面提供了 Lakehouse 命令行客户端、JDBC 驱动、生态工具 Connector 以及编程语言 SDK 的下载地址。

* 命令行客户端：[安装包下载地址](https://autolake-dev-beijing.oss-cn-beijing.aliyuncs.com/clickzetta-tool/release/sqlline_cz.tar.gz)
* [Java SDK（含JDBC驱动）](java_reference/java-sdk-summary.md)：[Maven库下载地址](https://repo1.maven.org/maven2/com/clickzetta/clickzetta-java)（Lakehouse 的 Java SDK 和 JDBC 驱动为同一个包）
* [SQLAlchemy](sqlalchemy.md)：PyPI 公开库下载 `pip install clickzetta-sqlalchemycata`
* [Python SDK](python_reference/connector.md)：PyPI 公开库下载 `pip install clickzetta-connector`
* [Zettapark](ZettaparkQuickStart.md)：PyPI 公开库下载 `pip install clickzetta_zettapark_python`
