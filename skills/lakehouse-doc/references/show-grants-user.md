## 功能

此命令用于查询指定用户在当前数据库中拥有的权限，帮助用户了解其在数据库中的访问和操作权限。

## 语法

```SQL
-- 查询指定用户拥有的权限
SHOW GRANTS TO USER user_name;
```

**参数说明：**

- `user_name`：需要查询权限的用户名称。该用户必须存在于当前数据库空间下。

## 示例

1. 查询当前登录用户的权限：

   ```SQL
   SHOW GRANTS TO USER user_name;
   ```

2. 查询指定用户 "john_doe" 拥有的权限：

   ```SQL
   SHOW GRANTS TO USER john_doe;
   ```

## 注意事项

- 执行此命令的用户需要具备查询权限。
- 在查询其他用户的权限时，请确保使用正确的用户名。
- 请谨慎对待权限管理，避免泄露敏感信息或误操作。

## 常见问题

**Q: 如何查询当前登录用户？**

A: 使用 `CURRENT_USER()` 函数代替 `user_name` 参数
