# 功能描述

`DESCRIBE HISTORY`命令用于获取表、动态表或物化视图的历史记录信息。通过这些历史记录，用户可以执行回滚操作，或者查询特定时间点的表数据。是否可以查看对象的操作历史，取决于数据的[保留周期](TIMETRAVEL.md)。

# 语法格式

```SQL
DESCRIBE HISTORY object_name;
```

参数说明：

* `object_name`：支持的类型包括表（table）、动态表（dynamic table）和物化视图（materialized view）。

# 注意事项

* 使用 `DESCRIBE HISTORY` 命令时，请确保 `object_name` 是有效的表、动态表或物化视图名称。
* 在查询特定时间点的数据时，请确保提供的时间格式正确。
