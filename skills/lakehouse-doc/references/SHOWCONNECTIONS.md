# 功能

本命令用于查询当前 Workspace 中的所有 CONNECTION 对象。

# 语法

```SQL
SHOW CONNECTIONS [ LIKE '<pattern>' | WHERE <expr>];
```

## 参数说明
1. `LIKE pattern`：此选项为可选参数，用于按对象名称进行过滤。支持不区分大小写的模式匹配，并可使用SQL通配符`%`（表示任意数量的字符）和`_`（表示单个字符）。示例：`LIKE '%testing%'`。需要注意的是，此过滤器不支持与 `WHERE` 条件同时使用。


2. `WHERE expr`：此选项为可选参数，支持用户根据 `SHOW CONNECTIONS` 命令显示的字段进行筛选。用户可以通过表达式对结果进行过滤，以便更精确地查找所需的数据库对象。支持过滤的字段有：
  `name`、`category`、`type`、`enabled`、`created_time`


# 使用示例

1. 查询当前工作区内的所有CONNECTION：

   ```SQL
   SHOW CONNECTIONS;
   +------------------------+----------+----------------+---------+-------------------------+
   |          name          | category |      type      | enabled |      created_time       |
   +------------------------+----------+----------------+---------+-------------------------+
   | my_funciton_connection | API      | CLOUD_FUNCTION | ENABLED | 2024-12-31 17:36:05.104 |
   +------------------------+----------+----------------+---------+-------------------------+

   ```

2. 根据 WHERE 条件过滤：

   ```
   SHOW CONNECTIONS WHERE created_time>TIMESTAMP'2024-12-30 17:36:05.104';
+------------------------+----------+----------------+---------+-------------------------+
|          name          | category |      type      | enabled |      created_time       |
+------------------------+----------+----------------+---------+-------------------------+
| my_funciton_connection | API      | CLOUD_FUNCTION | ENABLED | 2024-12-31 17:36:05.104 |
+------------------------+----------+----------------+---------+-------------------------+
```
