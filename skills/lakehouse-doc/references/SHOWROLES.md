# 功能

本命令用于查询当前数据库空间下的所有角色信息。通过执行该命令，用户可以方便地了解和管理数据库中的权限分配情况。

# Workspace 角色管理

```SQL
SHOW ROLES [LIKE pattern];
```

## 参数说明

- **LIKE pattern**（可选）：此参数用于按角色名称进行过滤，只显示符合给定模式的角色信息。该过滤器采用不区分大小写的模式匹配，支持 SQL 通配符 `%` 和 `_`。例如，使用 `LIKE '%admin%'` 将只显示包含 "admin" 文本的角色信息。

## 使用示例

1. 查询当前空间下的所有角色：

   ```SQL
   SHOW ROLES;
   ```

2. 查询包含 "admin" 文本的角色名称：

   ```SQL
   SHOW ROLES LIKE '%admin%';
   ```

3. 查询特定角色的详细信息：

   ```SQL
   SHOW ROLES LIKE 'role_name';
   ```

   其中，`role_name` 为需要查询的角色名称。

## 注意事项

- 在执行该命令时，确保您具有足够的权限来查询角色信息。

通过以上说明和示例，用户可以更加方便地了解和管理数据库中的角色信息。在实际应用中，可以根据需要灵活使用 LIKE 过滤条件，以便更高效地获取所需数据。
