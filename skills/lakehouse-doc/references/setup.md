# 开始之前

当您的云器 Lakehouse 账户设置完成后，您可以通过以下任意一种方式访问云器 Lakehouse：

* [Lakehouse Studio](studio_manual.md)：通过基于浏览器的 Web 界面，使用我们全面集成的数据开发和管理工具包。
* [Lakehouse CLI](connect-with-cli.md)：云器 Lakehouse 命令行客户端。
* 应用程序：使用云器 Lakehouse 连接器和驱动程序构建的应用程序，以及第三方客户端工具和应用程序也受支持（请参阅连接到 [云器 Lakehouse 的应用程序和工具](tutorial_connect_to_lakehouse.md)）。

如果您尚未拥有账户，可以 [进行注册](LoggingIn.md) 获取账户。

有关定价和服务详情，请参阅 [定价页面](pricing.md)。

## 浏览器要求

云器 Lakehouse Studio 推荐使用 Google Chrome。其他浏览器未经过像 Chrome 那样广泛的测试，可能会出现一些意外行为。如果您在使用任何浏览器的 Web 界面时遇到问题，请联系 [云器 Lakehouse 支持团队](https://www.yunqi.tech/)。

## 云器 Lakehouse CLI 操作系统平台要求

云器 [Lakehouse CLI（命令行客户端）](connect-with-cli.md) 可以在以下平台上安装：

* Red Hat Enterprise Linux 或兼容的操作系统。
* macOS（64 位）。
* Microsoft Windows（64 位）。

目前尚未测试其他平台，可能与 Lakehouse CLI 不兼容。例如，某些 Linux 发行版可能默认缺少 Lakehouse CLI 所需的库。
