# 通过RATH对ClickZetta数据进行可视化分析

## 1. RATH简介

RATH不仅是一款功能强大的开源数据分析和可视化工具，可以与Tableau等知名工具相媲美，而且它通过增强分析引擎自动发现数据模式、见解和因果关系，从而实现探索性数据分析工作流程的自动化。RATH可以轻松实现多维数据可视化。

主要功能如下：

* 🤖 自动化数据洞察：一键获取数据见解！增强分析引擎可帮助您发现数据中的模式、见解和因果关系。这是一种全自动的方式，只需单击一下，即可探索和可视化数据集。

* 🛠 数据探索副驾驶：RATH将作为您的数据科学助手，了解您的意图并生成相关建议。

* 自然语言界面：用自然语言提出问题，从数据中获得答案和可视化。

* AutoVis：RATH将为您选择的数据生成最佳可视化效果。这使您可以专注于数据和变量，而无需担心如何实现可视化。

* 👓 数据管理器：自动数据管理器，用于生成数据和数据转换的摘要。

* 🎨 数据绘制工具：一种交互式、直观且强大的工具，通过直接为数据着色并具备进一步的分析功能来进行探索性数据分析。观看[此视频](https://youtu.be/djqePNyhz7w)了解如何使用数据绘制工具发现数据洞察。

* 📊 仪表板：构建一个漂亮的交互式数据仪表板（包括一个自动化的仪表板设计器，可以为您的仪表板提供建议）。

* 因果分析：识别并检查变量之间的因果关系，这可以帮助您探索数据、创建更好的预测模型并做出明智的业务决策。

## 2. 环境准备

1. 将源代码克隆到本地：
   ```
   git clone <https://github.com/clickzetta/Rath.git>
   ```

2. 进入RATH所在目录。

3. 启动Docker容器：
   ```
   docker-compose up
   ```

4. 打开浏览器并访问：[http://localhost:9083](http://localhost:9083)。

![环境准备图示](.topwrite/assets/image_1697461437518.png)

## 3. 从ClickZetta Lakehouse导入数据

1. 选择Connection类型为ClickZetta。

2. 按照以下格式填写连接字符串：
   ```
   clickzetta://yourusername:yourpassword@yourinstance.service/yourworkspace?vcluster=yourvcluster&schema=yourschema
   ```

![连接字符串图示](.topwrite/assets/image_1697462002448.png)

## 4. 查看数据档案

在RATH中，您可以查看数据档案以了解数据集的结构和内容。点击左侧导航栏中的“数据档案”，您将看到所有导入的数据集列表。

![数据档案图示](.topwrite/assets/image_1697462239382.png)

## 5. 进行可视化分析

在RATH中，您可以轻松创建各种类型的可视化图表来分析数据。以下是一些使用示例：

* 示例1：创建条形图以显示不同类别的销售额。
  1. 点击“新建分析”按钮。
  2. 选择数据集中的销售额和类别字段。
  3. 选择条形图作为可视化类型。
  4. 拖放字段到相应的轴和值区域。
  5. 调整图表样式和颜色。

* 示例2：创建折线图以显示销售额随时间的变化趋势。
  1. 点击“新建分析”按钮。
  2. 选择数据集中的销售额和日期字段。
  3. 选择折线图作为可视化类型。
  4. 拖放字段到相应的轴和值区域。
  5. 调整图表样式和颜色。

* 示例3：创建散点图以显示两个变量之间的关系。
  1. 点击“新建分析”按钮。
  2. 选择数据集中的两个需要分析的字段。
  3. 选择散点图作为可视化类型。
  4. 拖放字段到相应的轴和值区域。
  5. 调整图表样式和颜色。

![可视化分析图示](.topwrite/assets/image_1697462345560.png)

通过以上步骤，您可以轻松地使用RATH对ClickZetta数据进行可视化分析。无论是发现数据中的模式、见解，还是探索因果关系，RATH都能为您提供强大的支持。
