# 实名认证

实名认证是通过验证您的真实身份信息，以确认您创建的云器账号归属的过程。分为个人认证和企业认证两种类型。

| 类型   | 个人实名认证     | 企业实名认证                  |
| ------ | ------------ | ------------------------- |
| 适用对象 | 个人         | 企业、个体工商户、政府、事业单位、学校、组织等 |
| 账号归属 | 个人         | 企业                      |
| 认证方式 | 个人身份证+人脸识别 | 企业证件+法人人脸识别             |
| 认证耗时 | 10分钟       | 10分钟                    |

## 功能权限

对于有多用户的账户，实名认证仅能由拥有账户管理员权限的用户完成。创建该账户的第一个用户会被默认授予账户管理员角色。非管理员用户可查看认证状态，但无法操作认证。

## 如何查看认证状态？

登录云器平台，进入账户主页即可查看当前实名认证状态，如下：

## 企业实名认证

进入账户主页 -> 选择“认证升级” -> 选择“企业认证”，进入企业实名认证界面。认证流程分为两个步骤：填写企业基础信息 -> 企业法人完成人脸识别，如下图所示：
![](.topwrite/assets/image_1747638456789.png)
![](.topwrite/assets/image_1747638493466.png)
![](.topwrite/assets/image_1747638545329.png)

* 需提供的企业信息包括：营业执照、公司名称、统一社会信用代码、法人姓名、法人身份证号、开户行、开户账号。
* 营业执照支持 JPG/JPEG/PNG/PDF 格式，且文件大小小于 10MB；
* 公司名称、统一社会信用代码、法人姓名请保证与营业执照中的信息一致。

确认信息填写无误后，点击“下一步”。当企业信息校验通过后，进入法人人脸识别环节，系统将生成人脸识别二维码。法人使用手机扫码，在手机端完成人脸识别即可，如下图：
![](.topwrite/assets/image_1747639041666.png)

## 个人实名认证

进入账户主页 -> 选择“认证升级” -> 选择“个人认证”，进入个人实名认证界面。认证流程分为两个步骤：填写个人身份信息 -> 完成个人人脸识别，如下图：
![](.topwrite/assets/image_1747639086741.png)
![](.topwrite/assets/image_1747639102052.png)
![](.topwrite/assets/image_1747639142744.png)

* 需提供的个人信息包括：姓名、身份证号。

确认姓名、身份证号匹配且正确后，点击“下一步”，进入个人人脸识别环节。通过手机扫描二维码，在手机端完成人脸验证，如下图：
![](.topwrite/assets/image_1747639179450.png)

## 其他说明

1. 人脸识别可使用任何支持扫描二维码的移动设备完成，建议使用支付宝或微信。

2. 单账号每日仅支持认证3次。如您在认证过程中遇到问题，请联系 400-6767-862。

3. 实名认证信息不支持线上变更，具体包括：不支持将个人认证账户变更为企业认证账户；不支持变更已完成企业实名认证的企业信息。如有需要，请联系商务。
