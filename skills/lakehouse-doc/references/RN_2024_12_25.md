# 2024年12月25日 Lakehouse Studio 产品更新发布说明

本次发布（Release 2024.12.25, v1.6）引入了一系列新功能、增强特性以及修复项。请注意，这些更新将分阶段逐步推送至以下区域，更新将在发布日起一周至两周内完成，具体时间依据您所在的区域而定。

* 腾讯云北京区
* 腾讯云上海区
* 腾讯云广州区
* 阿里云上海区
* 阿里云新加坡区
* AWS 北京区
* AWS 新加坡区

## 不兼容改动

* 无

## 新功能特性

* 产品全局：提供新用户引导弹窗，结合功能引导体验和帮助文档，帮助用户了解和使用产品核心能力。
* 数据同步：新增 Redshift 数据源，支持离线数据的导入。
* 数据同步：实时同步单表停止增量同步后，恢复时支持按给定过滤条件进行同步，以降低同步开销。
* 数据同步：离线数据集成新增支持同步源端 S3 上 Parquet 格式的文件。

## 优化改进

* 产品全局：全新视觉升级，升级范围包括产品首页、数据资产、运维中心等模块。
* 任务开发：调度配置中的运行超时时长，优化为以分钟为单位。
* 运维中心：调度定时的 Cron 调度表达式展示调整为 24 小时制。
* 数据上传：选择本地文件上传至新表时，系统识别本地文件的名称后，默认将其填充为新建表名。
* 数据上传：数据写入方式的权限点限制增强。追加写入需具备 insert 和 update 权限；先清空后写入需具备 delete、insert 和 update 权限。

## 缺陷修复

* 任务开发：修复进入任务开发时，因未找到可用集群而报错提示的问题。
