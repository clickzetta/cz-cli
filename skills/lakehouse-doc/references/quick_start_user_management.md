# 入门指南：如何快速新增和管理用户

## 适用场景

完成产品[注册开通](LoggingIn.md)之后，您将获得一个具备全局最高权限的初始账号，通常称之为“账户管理员”或“管理员”。您可以使用这个账号来访问产品的所有功能。如果您需要与同事协作，并为不同人员创建并分配各自专属的账号，请阅读本指南。否则，可以跳过本文档，直接参考以下指南开始使用产品：

* [如何快速运行一个SQL](quick_start_sql_query.md)
* [如何快速上传导入数据](quick_start_upload_data.md)

## 操作指南

1. 登录产品后，您默认会自动进入“管理中心”的“账户主页”，如下图所示：

   ![](.topwrite/assets/image_1747990064729.png)

2. 通过管理中心的左侧导航栏，进入“账户管理 > 用户管理”，即可进入用户管理页面，如下图所示：

   ![](.topwrite/assets/image_1747990075720.png)

3. 点击页面右上角的“新建”按钮，选择“新建用户”，即可开始创建用户账号。

   ![](.topwrite/assets/image_1747990090897.png =297)

   > 备注：“自定义服务用户”是为自动化工具、任务和应用程序使用而创建的身份，无法在页面登录，此处可暂时忽略。如果想了解更多，详见：[服务用户管理](account_user_management.md)

4. 在弹窗中按照页面指引填写必填信息，即可创建新的账号。请注意以下两点：

   * 请记住在此设置的密码，后续需要将用户名和密码提供给该用户使用。
   * 手机号需要填写该账号使用者的手机号，以便后续的告警消息等能准确地推送给该用户。
     ![](.topwrite/assets/image_1747990274287.png)

5. 新的用户账号创建好后，需要向该用户提供以下两条信息：

   * 用户名和密码。
   * 账户登录地址，其格式为：<账户名>.accounts.clickzetta.com。您也可以直接复制下图所示的浏览器地址，发送给该用户。

     ![](.topwrite/assets/image_1747990303813.png)

6. 获得用户名和密码的用户，登录产品后，如需修改初始密码，可通过左侧导航栏进入“账户管理 > 用户管理”页面，找到对应账号，在操作菜单中选择“修改密码”即可。

   ![](.topwrite/assets/image_1747990312625.png)
