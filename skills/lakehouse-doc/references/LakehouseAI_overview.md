# Lakehouse AI 功能概述：

云器 Lakehouse AI 是一套集成在数据湖仓平台中的智能分析套件，旨在帮助企业充分挖掘数据价值，实现从数据存储到智能决策的完整闭环。通过将非结构化数据管理、多模态检索、AI 外部函数、Python 开发框架、知识库服务、对话式分析等能力原生集成到 Lakehouse 架构中，用户可以在统一的平台上完成数据发现、模型调用、预测分析等全流程智能化操作。

*   **非结构化数据发现与管理**：自动识别和索引文档、图片、音视频等非结构化数据，实现统一元数据管理和智能分类
*   **AI 外部函数**：将预训练模型封装为 SQL 函数，在查询中直接调用 AI 能力，简化机器学习应用开发
*   **多模态检索**：支持文本、图像、向量等多种数据类型的统一搜索，实现跨模态的智能语义检索
*   **AI + BI 统一工作流**：将传统的数据处理流程与 AI 数据应用流程无缝整合在 Lakehouse 平台上，实现真正的数据智能融合。
*   **Python 语言接口（Zettapark）**：提供类 Python 开发框架，支持数据工程师使用熟悉的语言进行大数据处理。
*   **知识库服务（邀测）**：提供基于向量数据库的企业知识管理平台，支持文档解析、语义搜索和智能问答的 API。
*   **Lakehouse 对话式分析 DataGPT**：通过自然语言对话完成数据查询和分析，让业务人员零门槛使用数据。
*   **Lakehouse MCP Server（邀测）**：与 AI 生态融合的指令即服务功能，让 AI 助手直接操作 Lakehouse 数据资源。
