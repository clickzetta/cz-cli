# 服务实例首页

在 Lakehouse 的服务实例首页，您可以快速了解其核心产品能力，查看您加入的工作空间，并轻松进入各种功能场景。以下是详细的使用指南：

## 如何进入服务实例首页

登录云器管理中心后，点击 Lakehouse 的服务实例按钮，即可进入 Lakehouse 服务实例首页。
![](.topwrite/assets/image_1735618364104.png)

在这里，您可以找到各种功能入口，以便快速开始开发和运维工作。
![](.topwrite/assets/image_1735618407214.png)

### 用户实例信息

在这一区域，您可以查看您的登录名称、所在实例 ID、当前所选的云服务商和地域信息。这有助于您了解当前的操作环境。

### 核心功能区域

平台为您提供了以下快捷入口，以便快速进入相关功能模块：

* **上传数据**：点击后弹出 Web 可视化数据上传界面，支持您将本地文件快速上传至云器 Lakehouse 系统。更多数据上传操作可参考 [使用数据上传功能上传数据](upload_data.md)。
* **同步数据**：云器 Lakehouse 提供了多种数据同步能力，支持您将不同类型的数据源稳定、高效地同步至云器 Lakehouse 系统。
* **查询 SQL**：点击进入数据开发模块。在该模块，您可以定义并运行周期任务、进行数据分析，以及使用运维监控等功能。
* **管理集群**：点击后进入云器 Lakehouse 的集群管理页面。

### 数据资产信息

* 数据存储：在此区域，您可以查看当前实例下存储的对象总数以及存储总量，帮助您了解数据规模。
* 计算用量：查看当前实例下近 7 天每日的计算用量，以便您合理分配资源。

**注意**：数据资产信息仅限实例管理员角色查看。

### 帮助文档

点击此处，您将进入云器 Lakehouse 的产品帮助文档平台，了解更多关于产品功能与操作指南的信息。

### 最近操作区域

该区域展示了您最近访问过的 10 个任务和数据对象。点击后可快速进入对应的任务或数据对象的操作页面。
