# OSS 数据源配置指南

## 概述

OSS（Object Storage Service）是阿里云提供的对象存储服务，适用于大规模、高可靠性的数据存储需求。通过配置 OSS 数据源，您可以实现数据的高效上传、下载和同步，支持多种数据使用场景。

## 参数配置

配置 OSS 数据源时，需要提供以下信息以确保成功连接到 OSS 服务：

* **数据源名称**：为您的 OSS 数据源指定一个唯一且易于识别的名称，例如 `OSSDataSync`。
* **阿里云OSS地址**：提供 OSS 服务的 Endpoint 连接地址。例如，`oss-cn-shanghai-internal.aliyuncs.com`。
* **阿里云 AccessKeyId**：阿里云账户的 AccessKey ID，用于身份验证。
* **阿里云 AccessKeySecret**：与 AccessKey ID 对应的私密访问密钥。
* **阿里云 OSS Bucket**：指定要连接的 OSS 存储桶名称，例如 `my-oss-bucket`。
* **数据源描述**：（可选）为数据源添加描述性信息，以帮助您或其他管理员理解该数据源的用途或特点。

## 连接配置

在连接配置方面，需要注意以下事项：

* **直连**：确保您输入的连接信息在公网可访问。如果源端开启了 IP 访问白名单，请确保数据集成服务的出口 IP 地址已被添加到白名单中，具体 IP 地址请联系技术支持人员。同时支持使用 OSS 内网或公网 Endpoint 进行访问，具体访问方式根据所配置的“阿里云 OSS 地址”域名决定。

## 注意事项

* 确保所有提供的连接信息准确无误，并且 OSS 服务是可访问的。
* 保护您的凭证信息，避免泄露给未经授权的人员。
* 定期检查并更新您的数据源配置，以适应存储桶结构的变化或新的安全要求。
* 监控数据同步任务的运行状态，以便及时发现并解决问题。

完成配置后，即可在数据同步任务中选择此 OSS 数据源，进行数据的导入或导出操作。通过直连方式，您可以实现快速的数据传输，提高数据处理效率。
