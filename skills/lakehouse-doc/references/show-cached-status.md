## 功能
`SHOW CACHED STATUS`命令用于查看当前数据库中被缓存的表状态。需要注意的是，此命令仅能显示通过`CACHE`命令手动添加到缓存中的表，而无法显示系统自动缓存的表。

## 语法
```
SHOW CACHED STATUS;
```

## 说明
- 使用`SHOW CACHED STATUS`命令后，将列出所有正在被缓存的表及其相关信息。
- 该命令不会显示因系统自动缓存机制而缓存的表，仅展示用户通过`CACHE`命令显式添加的表。

## 示例
```
-- 将tpch100g分区表part和nation添加到缓存中
CACHE TABLE tpch100g.part, nation;

-- 查看当前缓存的表状态
SHOW CACHED STATUS;
```

执行上述命令后，将显示类似于以下的结果：
```
+----------------+--------+-------+
| Table           | Refs   | Cache |
+----------------+--------+-------+
| tpch100g.part  | 10     | YES   |
| tpch100g.nation| 5      | YES   |
+----------------+--------+-------+
```

在此结果中，`Table`列显示表的名称，`Refs`列显示表的引用次数，`Cache`列显示表是否正在被缓存。
