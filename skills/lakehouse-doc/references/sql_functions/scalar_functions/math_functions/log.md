# LOG 函数

## 1. 功能描述

`LOG` 函数用于计算指定数值的自然对数（以 `e` 为底）或指定底数的对数。它能够将一个正数转换为对应的对数值，广泛应用于数学计算、数据分析和科学领域。通过指定底数，`LOG` 函数还可以计算任意底数的对数，满足多样化的计算需求。

## 2. 语法

`LOG` 函数的语法如下：

```sql
LOG(base, number)
```

### 参数说明：

* `number`：必须为正数，表示要计算对数的数值。
* `base`（可选）：指定对数的底数。如果不指定，则默认为自然对数（以 `e` 为底）。

### 返回值：

* 返回值为 `number` 的对数值，类型为浮点数。

## 3. 示例

^

计算指定底数的对数

```sql
SELECT LOG(2, 8) AS log_base_2;
+------------+
| log_base_2 |
+------------+
| 3.0        |
+------------+
```
