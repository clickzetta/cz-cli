### 幂运算函数：POW

#### 功能描述
`POW` 函数用于计算一个数的指定次方，即求解 `expr1` 的 `expr2` 次幂。该函数在数学、科学和工程计算等领域有广泛的应用。

#### 语法
```
POW(expr1, expr2)
```
#### 参数说明
* `expr1`：基数，类型为双精度浮点数（double）。
* `expr2`：指数，类型为双精度浮点数（double）。

#### 返回结果
返回值类型为双精度浮点数（double），表示 `expr1` 的 `expr2` 次方的计算结果。

#### 使用示例
1. 计算 2 的 3 次方：
   ```sql
   SELECT POW(2, 3); -- 结果为 8.0
   ```
2. 计算 5 的 2 次方：
   ```sql
   SELECT POW(5, 2); -- 结果为 25.0
   ```
3. 计算 3 的 0.5 次方（即求平方根）：
   ```sql
   SELECT POW(3, 0.5); -- 结果为 1.7320508075688772
   ```
4. 计算 10 的 5 次方除以 2 的 3 次方：
   ```sql
   SELECT POW(10, 5) / POW(2, 3); -- 结果为 12500.0 
   ```

#### 注意事项
* 当 `expr2` 为负数时，表示求解 `expr1` 的负指数幂。例如，`POW(2, -3)` 计算的是 1/(2³)。
* 当 `expr2` 为 0 时，对于任何非零的 `expr1`，结果都是 1。如果 `expr1` 为 0，则结果未定义。
* 当 `expr1` 为负数且 `expr2` 为非整数时，结果可能是一个复数。在某些数据库系统中，这可能导致错误或返回特定的复数表示。请根据所使用的数据库系统查阅相关文档以确保正确处理。
