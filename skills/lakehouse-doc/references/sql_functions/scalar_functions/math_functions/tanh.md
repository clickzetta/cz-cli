### TANH 函数
```sql
TANH(expr)
```
#### 功能描述
TANH 函数用于计算给定数值表达式 `expr` 的双曲正切（hyperbolic tangent），返回结果为双曲正切值。

#### 参数说明
* `expr`：待计算的数值表达式，类型为浮点数（double）。

#### 返回结果
返回计算结果的双曲正切值，类型为浮点数（double）。

#### 使用示例
```sql
-- 计算 0 的双曲正切值
> SELECT TANH(0);
0.0

-- 计算 1 的双曲正切值
> SELECT TANH(1);
0.7615941559557649

-- 计算 -1 的双曲正切值
> SELECT TANH(-1);
-0.7615941559557649

-- 计算一个负数值 -2 的双曲正切值
> SELECT TANH(-2);
0.9640275800758169 

-- 计算一个较大正数值 5 的双曲正切值
> SELECT TANH(5);
0.9999092042625951

-- 计算一个较小负数值 -0.5 的双曲正切值
> SELECT TANH(-0.5);
-0.46211715726
