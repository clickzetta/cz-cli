### 余弦函数 (COS)
```sql
COS(expr)
```
#### 功能描述
余弦函数 `COS` 用于计算给定角度（参数 `expr` ）的余弦值。该函数在数学、物理和工程等领域中有广泛应用，尤其在解决与周期性变化相关的问题时非常实用。

#### 参数说明
- `expr` (double 类型): 需要计算余弦值的角度，单位为弧度。如果传入的角度是度数，需要先将其转换为弧度。可以通过乘以 π/180 将度数转换为弧度。

#### 返回结果
返回参数 `expr` 的余弦值，结果类型为 double。

#### 使用示例
1. 计算 0 弧度的余弦值：
   ```sql
   SELECT COS(0);
   // 结果：1.0
   ```
2. 计算 90 度（π/2 弧度）的余弦值：
   ```sql
   SELECT COS(PI()/2);
   // 结果：0
   ```
3. 计算 180 度（π 弧度）的余弦值：
   ```sql
   SELECT COS(PI());
   // 结果：-1
   ```
4. 计算 270 度（3π/2 弧度）的余弦值：
   ```sql
   SELECT COS(3*PI()/2);
   // 结果：0
   ```
5. 计算一个负角度（-π/4 弧度）的余弦值：
   ```sql
   SELECT COS(-PI()/4);
   // 结果：约 0.7071
   ```

通过以上示例，您可以了解如何使用余弦函数 `COS` 来计算不同角度的余弦值。在实际应用中，您可以根据需要计算任意角度的余弦值，以便解决各种与周期性变化相关的问题。
