### COT 函数
```
COT(expr)
```
#### 功能描述
COT 函数用于计算给定角度的余切值，即 cotangent。余切是正切的倒数，即 cot(x) = 1/tan(x)。该函数在三角函数计算中具有广泛的应用。

#### 参数说明
* expr (double 类型): 需要计算余切值的角度，单位为弧度。

#### 返回结果
返回类型为 double，表示给定角度的余切值。

#### 使用示例
1. 计算 1 弧度的余切值：
```sql
SELECT COT(1);
```
结果：
```
0.6420926159343306
```
2. 计算 π/4 (45°) 的余切值：
```sql
SELECT COT(PI()/4);
```
结果：
```
1.0000000000000002
```
3. 计算 0 弧度的余切值：
```sql
SELECT COT(0);
```
结果：
```
Infinity
```
4. 计算 -π/2 (-90°) 的余切值：
```sql
SELECT COT(-PI()/2);
```
结果：
```
-6.123233995736766E-17
```
#### 注意事项
* 当给定角度为 0 或 π (180°) 时，COT 函数将返回无穷大，因为正切函数在这些角度处的值未定义。
* 当给定角度为整数倍的 π (180°) 时，COT 函数将返回 0，因为正切函数在这些角度处的值为 0。
