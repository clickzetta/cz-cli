### SQRT 函数
```
sqrt(expr)
```
#### 功能描述
SQRT 函数用于计算指定数值表达式（expr）的平方根。该函数接受一个 double 类型的参数，并返回一个 double 类型的计算结果。

#### 参数说明
* `expr` (double 类型)：需要计算平方根的数值表达式。

#### 返回结果
返回一个 double 类型的数值，表示参数 expr 的平方根。

#### 使用示例
1. 计算 0.16 的平方根：
```sql
SELECT SQRT(0.16);
```
结果：
```
0.4
```

2. 计算 25 的平方根：
```sql
SELECT SQRT(25);
```
结果：
```
5.0
```

3. 计算 9.8596 的平方根（保留两位小数）：
```sql
SELECT ROUND(SQRT(9.8596), 2);
```
结果：
```
3.14
```
