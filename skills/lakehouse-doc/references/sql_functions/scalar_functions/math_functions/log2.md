### LOG2 函数
```sql
log2(expr)
```
#### 功能描述
LOG2 函数用于计算给定数值 `expr` 以 2 为底的对数。该函数在计算机科学、信息论和其他领域中有广泛应用，例如计算二进制数的位数或评估信息熵。

#### 参数说明
* expr (double 类型): 需要计算对数的数值。

#### 返回结果
返回 expr 以 2 为底的对数值，结果为 double 类型。

#### 使用示例
1.  计算 2 的以 2 为底的对数：
    ```sql
SELECT log2(2); -- 结果为 1.0
```
2.  计算 8 的以 2 为底的对数：
    ```sql
SELECT log2(8); -- 结果为 3.0
```
3.  计算 1024 的以 2 为底的对数：
    ```sql
SELECT log2(1024); -- 结果为 10.0
```
#### 注意事项
* 当 expr 为负数或非数值时，LOG2 函数将返回 NULL。
