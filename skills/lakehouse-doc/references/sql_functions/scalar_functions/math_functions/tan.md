### TAN 函数

#### 概述
`TAN` 函数用于计算给定角度的正切值。在数学中，正切值表示直角三角形中对边与邻边的比值。此函数在三角函数计算和角度转换等场景中具有广泛的应用。

#### 语法
```
TAN(expr)
```
#### 参数
- `expr`: 需要计算正切值的角度，类型为 `DOUBLE`。

#### 返回结果
返回计算结果的正切值，类型为 `DOUBLE`。

#### 使用示例
1.  计算 0 度的正切值：
    ```sql
   SELECT TAN(0);
   -- 结果：0.0
   ```
2.  计算 45 度的正切值：
    ```sql
   SELECT TAN(45);
   -- 结果：1.6197751905438615
   ```
3.  计算 90 度的正切值（结果为无穷大）：
    ```sql
   SELECT TAN(90);
   -- 结果：-1.995200412208242
   ```
4.  计算 -90 度的正切值（结果为负无穷大）：
    ```sql
   SELECT TAN(-90);
   -- 结果：1.995200412208242
   ```
5.  计算 180 度的正切值：
    ```sql
   SELECT TAN(180);
   -- 结果：1.3386902103511544
   ```
