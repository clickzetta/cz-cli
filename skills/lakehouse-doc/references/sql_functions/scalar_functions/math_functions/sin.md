### 正弦函数 (SIN)

#### 概述
正弦函数 (SIN) 是一个数学函数，用于计算给定角度的正弦值。在 SQL 语言中，我们可以使用 `sin` 函数来实现这一功能。该函数接受一个参数，并返回一个 `double` 类型的值。

#### 语法
```
sin(expr)
```

#### 参数
- `expr`：一个 `double` 类型的数值，表示需要计算正弦值的角度（以弧度为单位）。

#### 返回结果
返回一个 `double` 类型的值，表示给定角度的正弦值。

#### 使用示例

1.  计算 0 弧度的正弦值：
    ```sql
SELECT sin(0);
```
    结果：
    ```
0.0
```

2.  计算 π/2 弧度（即 90°）的正弦值：
    ```sql
SELECT sin(pi()/2);
```
    结果：
    ```
1.0
```

3.  计算 π 弧度（即 180°）的正弦值：
    ```sql
SELECT sin(pi());
```
    结果：
    ```
1.2246467991473532E-16
```

4.  计算 3π/2 弧度（即 270°）的正弦值：
    ```sql
SELECT sin(3 * pi()/2);
```
    结果：
    ```
-1.0
```

5.  计算一个负角度（-π/4 弧度，即 -45°）的正弦值：
    ```sql
SELECT sin(-pi()/4);
```
    结果：
    ```
-0.7071067811865475
```

#### 注意事项
- 确保输入的参数是一个有效的数值，否则可能会导致计算错误或返回 `NULL`。
- 当输入的角度值为负数时，正弦函数会返回相应的负值。
- 正弦函数在实际应用中可用于解决三角函数问题，以及分析振动和波动等现象。
