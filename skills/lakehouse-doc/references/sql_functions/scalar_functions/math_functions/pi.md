### PI 函数
#### 简介
`pi()` 函数用于返回圆周率常量 π 的值。该函数不需要任何参数。
#### 语法
```
pi()
```
#### 返回类型
该函数返回一个 double 类型的数值。
#### 使用示例
**示例 1：查询圆周率的值**
```
SELECT pi();
```
返回结果：
```
3.141592653589793
```

**示例 2：计算圆的面积**
已知半径为 5：
```
SELECT pi() * 5 * 5;
```
返回结果：
```
78.53981633974483
```

**示例 3：计算球的体积**
已知半径为 3：
```
SELECT 4/3 * pi() * 3 * 3 * 3;
```
返回结果：
```
113.09724459528809
```
#### 注意事项
- `pi()` 函数返回的 π 值是一个近似值，其具体精度取决于所使用的数据库系统。
- 在实际应用中，根据需要保留适当的小数位数。
