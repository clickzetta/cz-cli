### 函数名称
`degrees` 函数

### 功能描述
`degrees` 函数用于将弧度制数值转换为角度制数值。在数学和物理领域中，角度的度量通常使用弧度制或角度制。弧度制是一种更为通用的度量方式，而角度制则更符合人们的直觉。通过使用 `degrees` 函数，用户可以轻松地在两者之间进行转换。

### 参数说明
`expr` (double 类型): 需要转换的弧度制数值。

### 返回结果
返回转换后的双精度浮点数，即角度制数值。

### 使用示例
以下示例展示了如何使用 `degrees` 函数将不同弧度制数值转换为角度制数值。

**示例 1**：将 π 弧度转换为角度制数值。
```sql
SELECT degrees(3.141592653589793);
```
结果：
```
180.0
```

**示例 2**：将 2π 弧度转换为角度制数值。
```sql
SELECT degrees(6.283185307179586);
```
结果：
```
360.0
```

**示例 3**：将 -π 弧度转换为角度制数值。
```sql
SELECT degrees(-3.141592653589793);
```
结果：
```
-180.0
```
