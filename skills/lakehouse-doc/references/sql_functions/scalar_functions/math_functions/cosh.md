### 双曲余弦函数 (COSH)

#### 功能描述
`cosh` 函数用于计算给定数值表达式的双曲余弦值。双曲余弦函数是双曲函数的一种，与余弦函数类似，但适用于描述双曲线几何形状。

#### 语法
```
cosh(expr)
```
#### 参数说明
- `expr`: 需要计算双曲余弦的数值表达式，类型为 `double`。

#### 返回结果
返回计算结果的双曲余弦值，类型为 `double`。

#### 使用示例
1. 计算 0 的双曲余弦值：
   ```sql
   SELECT cosh(0);
   -> 1.0
   ```
2. 计算数值 -1 的双曲余弦值：
   ```sql
   SELECT cosh(-1);
   -> 1.5430806348152437
   ```
3. 计算数值 2 的双曲余弦值：
   ```sql
   SELECT cosh(2);
   -> 3.7621956910836314
   ```
4. 计算一个列中的每个元素的双曲余弦值，例如 `column_name`：
   ```sql
   SELECT cosh(column_name) FROM table_name;
   ```
   这将返回一个新列，其中包含 `table_name` 表中 `column_name` 列中每个元素的双曲余弦值。

#### 注意事项
- 对于非常大的数值，双曲余弦函数的计算结果可能会超出数值范围，导致溢出。在这种情况下，请确保对输入值进行适当的范围限制。

通过使用 `cosh` 函数，您可以轻松地在数据库查询中计算双曲余弦值，从而在数据分析和处理中应用双曲几何概念。
