### 函数名称
E

### 功能描述
e() 函数用于返回自然对数的底数，即数学常数 e 的值。

### 语法
```
e()
```

### 返回类型
返回值的类型为 double。

### 使用示例
以下为 e() 函数的使用示例：

```sql
-- 查询 e 的值
SELECT e();
```

执行上述查询后，将返回自然常数 e 的值，约为 2.718281828459045。
