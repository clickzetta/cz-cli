### EXP2 函数
```
exp2(expr)
```
#### 功能描述
EXP2 函数用于计算以 2 为底、expr 为指数的幂（即 2^expr）。该函数接受一个 double 类型的参数，并返回一个 double 类型的结果。

#### 参数说明
* expr (double 类型): 指数。

#### 返回结果
返回一个 double 类型的数值，表示 2 的 expr 次幂。

#### 使用示例
以下为一些使用 EXP2 函数的示例：

1.  计算 2 的 3 次幂：
    ```sql
SELECT exp2(3); -- 结果为：8.0
```

2.  计算 2 的 -5 次幂：
    ```sql
SELECT exp2(-5); -- 结果为：0.03125
```

3.  计算 2 的 0 次幂：
    ```sql
SELECT exp2(0); -- 结果为：1.0
```

4.  计算 2 的一系列指数次幂：
    ```sql
SELECT exp2(-2), exp2(1), exp2(5); -- 结果为：0.25, 2.0, 32.0
```
