### 余割函数（CSC）
```sql
csc(expr)
```
#### 功能描述
余割函数 `csc` 用于计算给定角度值 `expr` 的余割值。余割函数是正弦函数的倒数，即 `csc(x) = 1/sin(x)`。该函数在数学、物理和工程等领域中有广泛应用。

#### 参数说明
* `expr` (double 类型): 需要计算余割值的角度值，单位为弧度。

#### 返回结果
返回类型为 `double`，表示计算得到的余割值。

#### 使用示例
1. 计算角度值为 1 弧度的余割值：
```sql
SELECT csc(1);
```
结果：
```
1.1883951057781212
```
2. 计算角度值为 π/2 弧度（90°）的余割值：
```sql
SELECT csc(PI()/2);
```
结果：
```
1.0
```

#### 注意事项
* 当 `expr` 参数值为 0 时，余割函数的结果为无穷大（`Infinity`）。
* 余割函数仅接受以弧度为单位的角度值。如需将角度转换为弧度，请使用 `radians()` 函数。
