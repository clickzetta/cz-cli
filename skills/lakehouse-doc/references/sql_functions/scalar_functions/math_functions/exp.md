### EXP 函数
```
EXP(expr)
```
#### 功能描述
EXP 函数用于计算表达式 `expr` 的 `e` 次幂，即自然指数函数。`e` 是一个数学常数，大约等于 2.71828。

#### 参数说明
- `expr`：需要计算指数的数值，类型为双精度浮点数（double）。

#### 返回结果
返回类型为双精度浮点数（double），表示计算结果。

#### 使用示例
1. 计算 `e` 的 1 次幂：
   ```sql
   SELECT EXP(1.0); -- 结果约为：2.718281828459045
   ```
2. 计算 `e` 的 2 次幂：
   ```sql
   SELECT EXP(2.0); -- 结果约为：7.38905609893033
   ```
3. 模拟连续复利计算，假设本金为 1000 元，年利率为 5%，存款 3 年：
   ```sql
   SELECT 1000 * EXP(0.05 * 3); -- 结果约为：1161.834242728283 
   ```

#### 注意事项
- 当 `expr` 为负数时，EXP 函数返回的是一个小于 1 的正数。
- 当 `expr` 为 0 时，EXP 函数返回 1。
- 当 `expr` 为正无穷大时，EXP 函数返回正无穷大。
- 当 `expr` 为负无穷大时，EXP 函数返回 0。
