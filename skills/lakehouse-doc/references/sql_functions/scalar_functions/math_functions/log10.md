### LOG10 函数
```sql
LOG10(expr)
```
#### 功能描述
LOG10 函数用于计算给定数值表达式 `expr` 的以 10 为底的对数。该函数在处理指数型数据时非常有用，可以帮助快速了解数据的增长速度和规模。

#### 参数说明
* expr (double 类型): 需要计算对数的数值表达式。

#### 返回结果
返回计算结果为 double 类型。

#### 使用示例
1. 计算 100 的以 10 为底的对数：
```sql
SELECT LOG10(100);
```
结果：
```
2.0
```
2. 计算 1000 的以 10 为底的对数，并与常见对数进行比较：
```sql
SELECT LOG10(1000);
```
结果：
```
3.0
```
3. 计算不同数值的以 10 为底的对数，观察对数值的变化：
```sql
SELECT LOG10(1), LOG10(10), LOG10(100), LOG10(1000);
```
结果：
```
0.0 1.0 2.0 3.0
```
4. 计算连续整数的以 10 为底的对数，了解数值增长速度：
```sql
SELECT LOG10(1), LOG10(10), LOG10(100), LOG10(1000), LOG10(10000);
```
结果：
```
0.0 1.0 2.0 3.0 4.0
```
通过以上示例，可以观察到随着数值的增长，以 10 为底的对数值也随之增长。LOG10 函数在数据分析和处理中具有广泛的应用，可以帮助用户快速了解数据的规模和增长速度。
