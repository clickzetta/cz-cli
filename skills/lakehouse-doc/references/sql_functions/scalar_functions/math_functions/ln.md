### LN 函数

#### 功能描述
LN 函数用于计算给定数值表达式的自然对数（以 e 为底）。自然对数在数学、物理和工程等领域中有广泛的应用，例如在处理指数增长和衰减问题时。

#### 语法
```
LN(expr)
```

#### 参数
- `expr`: 需要计算自然对数的数值表达式，类型为 double。

#### 返回结果
返回计算结果的数值，类型为 double。

#### 使用示例
1. 计算一个常数值的自然对数：
   ```sql
   SELECT LN(3.1415926);
   ```
   结果：
   ```
   1.144729868791239
   ```

2. 计算一个负数的自然对数（结果为 NULL）：
   ```sql
   SELECT LN(-1);
   ```
   结果：
   ```
   null
   ```

3. 计算一个较大数值的自然对数：
   ```sql
   SELECT LN(1000000);
   ```
   结果：
   ```
    13.815510557964274
   ```

4. 计算 e 的自然对数（结果应接近 1）：
   ```sql
   SELECT LN(EXP(1));
   ```
   结果：
   ```
   1.0
   ```

#### 注意事项
- 当 `expr` 参数为负数时，LN 函数将返回 NULL，因为负数在实数域内没有自然对数。
- 当 `expr` 参数为 0 时，LN 函数也将返回 NULL，因为 0 没有自然对数。
- 在使用 LN 函数时，请确保传入的表达式为有效的 double 类型数值，否则可能导致计算结果不正确或返回 NULL。
