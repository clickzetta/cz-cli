### ROUND 函数

#### 功能描述
ROUND 函数用于对数值类型的表达式进行四舍五入操作，保留指定位数的小数。该函数支持多种输入数据类型，包括 float、double、decimal、smallint、tinyint、int 和 bigint。

#### 语法格式
```sql
ROUND(expr [, d])
```

#### 参数说明
- `expr`：需要进行四舍五入操作的数值类型表达式。
- `d`：用于指定保留小数位数的整数，支持负值。默认值为 0。

#### 返回结果
返回四舍五入后的数值结果，类型与输入的 `expr` 相同。

#### 使用示例

1. 保留一位小数：
   ```sql
   SELECT ROUND(3.14, 1); -- 结果为：3.1
   ```

2. 保留两位小数：
   ```sql
   SELECT ROUND(3.1415926, 2); -- 结果为：3.14
   ```

3. 保留整数部分：
   ```sql
   SELECT ROUND(2.5); -- 结果为：3
   ```

4. 保留整数部分（省略小数位数参数）：
   ```sql
   SELECT ROUND(314.1592, -1); -- 结果为：310
   ```

5. 对负数进行四舍五入：
   ```sql
   SELECT ROUND(-3.14, 1); -- 结果为：-3.1
   ```

6. 对 DECIMAL 类型进行四舍五入：
   ```sql
   SELECT ROUND(123.4567, 2); -- 结果为：123.46
   ```
