### ACOSH 函数
#### 概述
`ACOSH` 函数用于计算给定数值的反双曲余弦值。该函数接受一个 `DOUBLE` 类型的参数，并返回一个 `DOUBLE` 类型的数值结果。

#### 语法
```
ACOSH(expr)
```
#### 参数
- `expr`: 需要计算反双曲余弦值的 `DOUBLE` 类型数值。

#### 返回结果
返回参数 `expr` 的反双曲余弦值，结果类型为 `DOUBLE`。

#### 使用示例
1. 计算 `2` 的反双曲余弦值：
```
SELECT ACOSH(2); -- 结果约为 1.3169578969248166
```
2. 计算 `1` 的反双曲余弦值（结果为 `0`，因为 `1` 是 `ACOSH` 函数的极小值）：
```
SELECT ACOSH(1); -- 结果为 0.0
```
3. 计算接近于 `0` 的小数值的反双曲余弦值：
```
SELECT ACOSH(0.0001); -- 结果约为 NaN
```
4. 计算负数的反双曲余弦值（结果为 `NaN`，因为 `ACOSH` 函数只接受正数参数）：
```
SELECT ACOSH(-1); -- 结果为 NaN
```
5. 在一个查询中对多个数值计算反双曲余弦值：
```
SELECT ACOSH(2.5), ACOSH(3.7), ACOSH(0.5); -- 结果分别约为 1.566799236972411, 1.9826969446812033和 NaN
```
#### 注意事项
- `ACOSH` 函数仅接受正数参数。如果传入负数或零，将返回 `NaN`（非数值结果）。
- 当参数值接近于 `0` 时，反双曲余弦值会变得非常大。在实际应用中，需要注意数值溢出的问题。
- `ACOSH` 函数与 `COSH` 函数互为逆运算。`COSH` 函数用于计算双曲余弦值，而 `ACOSH` 函数用于计算其逆运算结果。
