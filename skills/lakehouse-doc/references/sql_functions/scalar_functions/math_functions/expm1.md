### 指数函数的补数（EXPM1）

#### 概述
`expm1` 函数用于计算 `exp(expr)-1` 的值，其中 `expr` 是一个 double 类型的参数。这个函数可以避免直接计算 `exp(x)` 时可能遇到的数值溢出或精度问题，特别适用于计算较小的指数值。

#### 功能
该函数计算 `e` 的 `expr` 次幂减去 1 的值，其中 `e` 是自然对数的底数（约等于 2.71828）。这个计算方法在数值稳定性方面具有优势，特别是在处理接近零的指数值时。

#### 参数
- `expr` (double 类型)：需要计算其指数补数的数值。

#### 返回结果
返回一个 double 类型的值，表示 `exp(expr)-1` 的结果。

#### 使用示例
以下为一些使用 `expm1` 函数的示例：

1.  计算 `exp(1)-1` 的值：
    ```sql
   SELECT expm1(1);
   ```
    结果：
    ```
   1.718281828459045
   ```
   
2.  计算 `exp(0.5)-1` 的值：
    ```sql
   SELECT expm1(0.5);
   ```
    结果：
    ```
   0.6487212707001282
   ```

3.  计算 `exp(-1)-1` 的值：
    ```sql
   SELECT expm1(-1);
   ```
    结果：
    ```
   -0.6321205588285577
   ```

4.  计算一系列指数补数的值：
    ```sql
   SELECT expm1(-0.1), expm1(0), expm1(0.1), expm1(1);
   ```
    结果：
    ```
   -0.09516258196404043, 0, 0.10517091807564763, 1.718281828459045
   ```
