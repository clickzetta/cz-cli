### CBRT 函数
#### 简介
CBRT 函数用于计算给定数值的立方根。该函数接受一个 double 类型的参数，并返回一个 double 类型的计算结果。

#### 语法
```
CBRT(expr)
```
#### 参数
- `expr`: 需要计算立方根的 double 类型数值。

#### 返回结果
返回参数 `expr` 的立方根，结果类型为 double。

#### 使用示例
1. 计算 27 的立方根：
```
SELECT CBRT(27.0);
```
结果：
```
3.0000000000000004
```
2. 计算一个负数的立方根（例如 -8）：
```
SELECT CBRT(-8.0);
```
结果：
```
-2.0
```
3. 计算一个带小数的数值的立方根（例如 0.027）：
```
SELECT CBRT(0.027);
```
结果：
```
0.29999999999999993
```
4. 计算一个较大数值的立方根（例如 1000000）：
```
SELECT CBRT(1000000);
```
结果：
```
100.0
```
5. 在一个查询中同时计算多个数值的立方根：
```
SELECT CBRT(27.0), CBRT(64.0), CBRT(-125.0);
```
结果：
```
3.0000000000000004, 4.0, -5.0
```
通过以上示例可以看出，CBRT 函数可以灵活地应用于各种场景，帮助用户快速计算出所需数值的立方根。
