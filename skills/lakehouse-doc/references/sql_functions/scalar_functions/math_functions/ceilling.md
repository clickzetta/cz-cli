#### 简介

`CEILING` 函数用于返回大于或等于指定数值的最小整数。换句话说，它将给定的数值向上舍入到最接近的整数。这在需要确保数值不低于某个特定值的情况下非常有用。

#### 语法

```Plain
CEILING(value)
```

#### 参数

* `value`：要向上舍入的数值，可以是整数、浮点数或其他数值类型。

#### 返回结果

* 返回大于或等于 `value` 的最小整数，结果的类型通常为整数。

#### 使用示例

1. **基本示例**：

```SQL
SELECT CEILING(3.2) AS result;
+--------+
| result |
+--------+
| 4      |
+--------+
```

2. **负数示例**：

```SQL
SELECT CEILING(-2.7) AS result;
+--------+
| result |
+--------+
| -2     |
+--------+
```

#### 注意事项

* `CEILING` 函数的返回值始终是一个整数类型。
* 对于正数，`CEILING` 函数会向上舍入到下一个整数；对于负数，它会返回较小的整数（即更接近零的整数）。
* 如果输入值已经是一个整数，`CEILING` 函数将返回该整数本身。
