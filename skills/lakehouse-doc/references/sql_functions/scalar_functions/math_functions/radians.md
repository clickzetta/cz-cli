### 弧度函数
```sql
radians(expr)
```
#### 功能描述
`radians` 函数用于将角度值转换为对应的弧度值。在数学中，角度和弧度是两种度量圆周的方式，其中弧度是更通用的度量单位。该函数可以确保计算的准确性和一致性。

#### 参数说明
- `expr`：要转换的角度值，类型为 `double`。

#### 返回结果
返回转换后的弧度值，类型为 `double`。

#### 使用示例
1. 将 90 度转换为弧度：
```sql
SELECT radians(90);
-- 返回结果：1.5707963267948966
```
2. 将 180 度转换为弧度：
```sql
SELECT radians(180);
-- 返回结果：3.141592653589793
```
3. 将 360 度转换为弧度：
```sql
SELECT radians(360);
-- 返回结果：6.283185307179586
```
