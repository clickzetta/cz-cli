### HYPOT 函数

#### 概述
HYPOT 函数用于计算两个数的平方和的平方根，即计算直角三角形的斜边长度。这个函数在计算几何问题时非常有用，尤其是在处理直角三角形时。

#### 语法
```
HYPOT(expr1, expr2)
```

#### 参数
- `expr1`: 一个 double 类型的数值表达式。
- `expr2`: 一个 double 类型的数值表达式。

#### 返回结果
返回一个 double 类型的数值，表示两个参数平方和的平方根。

#### 使用示例

1. 计算直角三角形的斜边长度：
   ```sql
   > SELECT HYPOT(3, 4);
   5
   ```
   在这个例子中，我们计算了一个直角三角形的斜边长度，其中两条直角边的长度分别为 3 和 4。HYPOT 函数返回斜边的长度，即 5。

2. 计算一个点到原点的距离：
   ```sql
   > SELECT HYPOT(x, y) FROM points WHERE x = 3 AND y = 4;
   5
   ```
   在这个例子中，我们从一个包含点坐标的表（points）中查询点 (3, 4) 到原点 (0, 0) 的距离。HYPOT 函数计算了点 (3, 4) 到原点 (0, 0) 的距离，返回结果为 5。


#### 注意事项
- 确保传入 HYPOT 函数的参数是数值类型的，否则可能导致函数返回错误或异常。
- 当两个参数都为 0 时，HYPOT 函数将返回 0，因为 0 的平方和的平方根仍然是 0。
