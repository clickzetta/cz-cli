### 函数名称
RANDN 函数

### 功能描述
RANDN 函数是一个非确定性函数，用于生成遵循标准正态分布（均值为 0，标准差为 1）的随机数。该函数在模拟数据、数值分析和统计测试等各种场景下具有广泛的应用。

### 返回类型
该函数返回一个 double 类型的值。

### 使用示例
1. 生成一个标准的正态分布随机数：
   ```sql
   SELECT RANDN();
   ```
   结果可能类似于：`0.635793277734877`

2. 生成多个正态分布随机数并计算平均值：
   ```sql
   SELECT AVG RANDN() FROM (SELECT NULL) AS x;
   ```
   结果可能类似于：`0.0`
   在这个例子中，我们通过生成随机数并计算其平均值，可以看到结果趋近于 0，符合标准正态分布的均值特性。


### 结论
RANDN 函数是一个简单而强大的工具，可以帮助用户在 SQL 查询中生成符合正态分布的随机数。通过灵活运用此函数，用户可以在数据分析和处理中实现更多的可能性。
