#### 功能描述

`median` 函数用于计算一组数值的中位数。中位数是将一组数值按大小顺序排列后，位于中间位置的数值。如果数值集合有偶数个元素，则中位数是中间两个数值的平均值。`median` 函数自动忽略 `NULL` 值。

#### 语法

```Plain
median(col)
```

#### 参数

* `col`: 需要计算中位数的列。

#### 返回结果

* 返回指定列的中位数。

#### 使用示例

```SQL
SELECT median(col) FROM VALUES (1), (2), (2), (3), (4), (NULL) AS tab(col);
```

在这个示例中，`median` 函数计算了由 `VALUES` 子句提供的一组数值的中位数。由于 `NULL` 值被忽略，实际参与计算的数值为 `1, 2, 2, 3, 4`。这组数值按大小顺序排列后为 `1, 2, 2, 3, 4`，中间位置的数值是 `2` 和 `3`，因此返回的中位数是这两个数值的平均值，即 `2.5`。
