### 除法运算符 DIV

#### 功能描述
DIV 运算符用于计算两个数值整除的结果。

#### 语法
```
divisor div dividend
```

#### 参数说明
- `divisor`: 被除数，只能是整数。
- `dividend`: 除数，只能是整数。

#### 返回类型
返回结果为 `bigint` 类型。

#### 使用示例

1. 整数除法：
   ```sql
   SELECT 10 div 3; -- 结果为 3
   SELECT -10 div 3; -- 结果为 -3
   ```

#### 注意事项
- 当除数为零时，除法运算无法执行，结果为 NULL。
