### HEX 函数
```sql
HEX(expr)
```
#### 功能描述
HEX 函数用于将不同类型的数据转换为十六进制格式的字符串。该函数接受多种输入类型，包括字符串、二进制数据和大整数，并将它们转换为对应的十六进制表示形式。

#### 参数说明
* **expr**: string/binary/bigint
  * 待转换的输入数据。

#### 返回结果
返回一个字符串，表示输入数据的十六进制形式。

#### 使用示例
1. 字符串转换为十六进制：
    ```sql
   SELECT HEX('HelloWorld');
   ```
    结果：
    ```
   48656C6C6F576F726C64
   ```
2. 二进制数据转换为十六进制：
    ```sql
   SELECT HEX(CAST('11001010' AS BINARY));
   ```
    结果：
    ```
   3131303031303130

   ```
3. 大整数转换为十六进制：
    ```sql
   SELECT HEX(1234567891234567);
   ```
    结果：
    ```
   462D53C9BAF07
   ```

#### 注意事项
- 对于空字符串输入，HEX 函数将返回空字符串。
- 当输入数据无法转换为十六进制时，函数将返回 NULL。
