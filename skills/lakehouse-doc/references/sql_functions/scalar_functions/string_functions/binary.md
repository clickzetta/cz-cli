### 二进制转换函数

#### 功能描述
`BINARY()` 函数用于将字符串类型的数据转换成二进制形式的字节流。该函数接受一个字符串参数，并返回一个二进制数据类型的结果。

#### 语法
```
BINARY(str)
```
#### 参数
- `str`: 需要转换的字符串类型的数据。

#### 返回结果
- 返回一个二进制数据类型的字节流。

#### 使用示例
1. 将字符串 "hello" 转换为二进制形式：
   ```sql
   SELECT BINARY('hello');
   ```
   结果：
   ```
    [68 65 6c 6c 6f] 
   ```
2. 将一个包含特殊字符的字符串 "élle!" 转换为二进制形式：
   ```sql
   SELECT BINARY('élle!');
   ```
   结果：
   ```
   [c3 a9 6c 6c 65 21]
   ```
3. 将一个中文字符串 "你好" 转换为二进制形式：
   ```sql
   SELECT BINARY('你好');
   ```
   结果：
   ```
   [e4 bd a0 e5 a5 bd]
   ```

#### 注意事项
- 当输入的字符串包含非ASCII字符时，`BINARY()` 函数会对其进行编码转换，以便在二进制形式中正确表示。
- 在处理二进制数据时，需要注意字符集和校对规则的设置，以确保数据的正确性和一致性。

通过使用 `BINARY()` 函数，您可以方便地在数据库中存储和操作二进制数据，例如图像、音频和视频文件等。
