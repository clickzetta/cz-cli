### UCASE 函数

```
ucase(str)
```

#### 功能描述

`UCASE` 函数是 `UPPER` 的别名，用于将输入字符串中的所有小写字母转换为大写字母，并返回转换后的字符串。

#### 参数说明

* `str`：需要转换为大写的字符串。

#### 返回类型

* 返回一个字符串，其中所有小写字母均已转换为大写字母。

#### 注意事项

* `UCASE` 函数与 `UPPER` 函数完全相同，可以互换使用。
* `UCASE` 函数仅适用于输入字符串中包含的英文字母。对于其他语言或特殊字符，转换效果可能不符合预期。

#### 使用示例

1. 转换简单字符串

```sql
SELECT UCASE('hello world');
+-----------------------+
| UCASE('hello world')  |
+-----------------------+
| HELLO WORLD           |
+-----------------------+
```

2. 转换包含数字和特殊字符的字符串

```sql
SELECT UCASE('Hello123!@#');
+-----------------------+
| UCASE('Hello123!@#')  |
+-----------------------+
| HELLO123!@#           |
+-----------------------+
```
