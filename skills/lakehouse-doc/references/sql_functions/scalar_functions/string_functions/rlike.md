### RLIKE 函数
#### 概述
RLIKE 函数用于检查一个字符串（str）是否符合给定的正则表达式（regex）。如果符合，则返回 true，否则返回 false。该函数使用 [re2](https://github.com/google/re2) 正则表达式引擎进行匹配。

#### 语法
```
str [not] rlike regex

rlike(str, regex)
```
#### 参数
* str (string)：待匹配的字符串。
* regex (string)：正则表达式。请注意，由于 SQL 解析器会对字符串常量进行转义，因此在使用常量指定正则表达式时需要使用双反斜杠（例如：'^\\\\abc$'）。

#### 返回结果
返回一个布尔值（boolean），表示 str 是否符合 regex。

#### 使用示例
1. 检查字符串 "100-500" 是否符合正则表达式 `\d+-\d+`（数字加短横线加数字）：
```sql
SELECT '100-500' RLIKE '\d+-\d+';
```
结果：
```
true
```

2. 使用函数形式检查字符串 "100-500" 是否符合正则表达式 `\d+-\d+`：
```sql
SELECT RLIKE('100-500', '\d+-\d+');
```
结果：
```
true
```

3. 检查字符串 "\\abc" 是否符合正则表达式 `^\\\\abc$`（匹配以 "abc" 开头和结尾的字符串，正则表达式本身可能包含反斜杠进行转义）：
```sql
SELECT RLIKE('\\abc', '^\\\\abc$');
```
结果：
```
true
```

4. 检查字符串 "abc" 是否**不符合**正则表达式 `^\\d+$`（以数字开头和结尾的字符串）：
```sql
SELECT 'abc' RLIKE '^\\d+$';
```
结果：
```
false
```

5. 使用 "not" 关键字检查字符串 "123" 是否**不符合**正则表达式 `\d+`（包含数字的字符串）：
```sql
SELECT '123' NOT RLIKE '\d+';
```
结果：
```
false
```

通过以上示例，您可以更好地理解 RLIKE 函数的用法和功能。在实际应用中，您可以根据需要使用不同的正则表达式来匹配和验证字符串。
