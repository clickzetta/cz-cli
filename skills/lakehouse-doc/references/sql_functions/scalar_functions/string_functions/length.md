### LENGTH 函数

#### 功能描述
LENGTH 函数用于计算字符串的长度，返回字符串的字符数或二进制字符串的字节数。该函数对于文本数据的处理非常实用，可以快速获取文本的长度信息。

#### 语法
```sql
LENGTH(expr)
```

#### 参数说明
- `expr`：可以是字符串（string）或二进制（binary）类型的表达式。

#### 返回结果
返回一个整数（int），表示输入字符串的字符数或二进制字符串的字节数。

#### 使用示例
1. 计算普通字符串的长度：
   ```sql
   SELECT LENGTH('HelloWorld'); -- 返回结果为 10
   ```
2. 计算包含中文字符的字符串长度：
   ```sql
   SELECT LENGTH('你好世界'); -- 返回结果为 4
   ```
3. 计算包含特殊字符的字符串长度：
   ```sql
   SELECT LENGTH('Hello@World!'); -- 返回结果为 12
   ```


#### 注意事项
- LENGTH 函数只适用于字符串和二进制类型的数据，对于其他类型的数据将返回错误。
