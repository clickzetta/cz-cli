## 功能概述

CHARACTER_LENGTH 用于返回字符串中字符的数量。

## 语法

```SQL
CHARACTER_LENGTH(string|char|)
```

## 参数说明

* **string**: 要计算字符长度的字符串表达式。

## 返回结果

该函数返回一个整数，表示输入字符串中的字符总数。

## 示例

示例 1：基本使用

```SQL
SELECT CHARACTER_LENGTH('Hello, World!');
+-------------------------------------+
| `CHARACTER_LENGTH`('Hello, World!') |
+-------------------------------------+
| 13                                  |
+-------------------------------------+
```

示例 2：处理 NULL 值

```SQL
SELECT CHARACTER_LENGTH(NULL);
+--------------------------+
| `CHARACTER_LENGTH`(NULL) |
+--------------------------+
| null                     |
+--------------------------+
```

示例 3：中文字符

```SQL
 SELECT CHARACTER_LENGTH('中文');
+--------------------------+
| `CHARACTER_LENGTH`('中文') |
+--------------------------+
| 2                        |
+--------------------------+
```
