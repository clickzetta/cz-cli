## 功能描述

`REGEXP_EXTRACT_ALL` 函数用于从字符串中提取与正则表达式匹配的所有子串。

## 语法

```SQL
REGEXP_EXTRACT_ALL(string, pattern)
```

## 参数说明

* **string**: 要搜索的原始字符串。
* **pattern**: 用于匹配的正则表达式。

## 返回结果

返回一个数组，包含所有匹配的子串。

## 示例

```SQL
SELECT regexp_extract_all('100-200, 300-400',r'(\d+)-(\d+)', 1);
+---------------+
|      res      |
+---------------+
| ["100","300"] |
+---------------+
```
