### UNHEX 函数
```
unhex(expr)
```
#### 功能描述
UNHEX 函数用于将十六进制格式的字符串转换为二进制数据。该函数接受一个字符串参数，并返回相应的二进制数据。

#### 参数说明
* `expr` (string): 需要转换的十六进制格式字符串。

#### 返回结果
返回二进制数据。

#### 使用示例
1. 将十六进制字符串转换为文本：
```sql
SELECT STRING(UNHEX('48656C6C6F576F726C64'));
```
结果：
```
HelloWorld
```
2. 将十六进制字符串转换为图片数据：
```sql
SELECT UNHEX('89504E470D0A1A0A0000') AS image_data;
```
结果：
```
 [89 50 4e 47 0d 0a 1a 0a 00 00]
```
3. 将十六进制字符串转换为音频数据：
```sql
SELECT UNHEX('25504446') AS audio_data;
```
结果：
```
 [25 50 44 46]
```
4. 将十六进制字符串转换为视频数据：
```sql
SELECT UNHEX('000001BA') AS video_data;
```
结果：
```
[00 00 01 ba]
```
