## 功能概述

CHAR\_LENGTH等效于CHARACTER\_LENGTH，用于返回字符串中字符的数量。

## 语法

```SQL
CHAR_LENGTH(string|char|)
```

## 参数说明

* **string**: 要计算字符长度的字符串表达式。

## 返回结果

该函数返回一个整数，表示输入字符串中的字符总数。

## 示例

示例 1：基本用法

```SQL
SELECT CHAR_LENGTH('Hello, World!');
+-------------------------------------+
| `CHAR_LENGTH`('Hello, World!') |
+-------------------------------------+
| 13                                  |
+-------------------------------------+
```

示例 2：与 NULL 值一起使用

```SQL
SELECT CHAR_LENGTH(NULL);
+--------------------------+
| `CHAR_LENGTH`(NULL) |
+--------------------------+
| null                     |
+--------------------------+
```

示例 3：中文字符

```SQL
 SELECT CHAR_LENGTH('中文');
+--------------------------+
| `CHARA_LENGTH`('中文') |
+--------------------------+
| 2                        |
+--------------------------+
```
