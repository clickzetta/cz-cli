### SPACE 函数

#### 功能描述
SPACE 函数用于生成一个由指定数量空格组成的字符串。该函数在需要对齐文本或填充空格时非常有用。

#### 语法
```
SPACE(n)
```

#### 参数
- **n**：bigint 类型，表示要生成的空格数量。

#### 返回值
- 返回一个包含 n 个空格的字符串。

#### 使用示例

1. 生成一个包含 5 个空格的字符串：
```sql
SELECT SPACE(5);
```


2. 在姓名中添加空格以生成完整的地址格式：
```sql
SELECT '姓名' || SPACE(10 - LENGTH('姓名')) || '街道地址' AS address;
```
结果：
```
姓名                街道地址
```

通过以上示例，您可以看到 SPACE 函数在不同场景下的应用。使用 SPACE 函数可以方便地生成指定长度的空格字符串，从而满足各种对齐和填充需求。
