### LCASE 函数

```
lcase(str)
```

#### 功能描述

`LCASE` 函数是 `LOWER` 的别名，用于将输入的字符串中的所有大写字符转换为小写字符，并返回转换后的字符串。

#### 参数说明

* `str`：需要转换为小写字母的字符串。

#### 返回类型

* 返回转换为小写字母的字符串。

#### 注意事项

* `LCASE` 函数与 `LOWER` 函数完全相同，可以互换使用。

#### 使用示例

1. 将 "HelloWorld" 转换为 "helloworld"

```sql
SELECT LCASE('HelloWorld');
+----------------------+
| LCASE('HelloWorld')  |
+----------------------+
| helloworld           |
+----------------------+
```

2. 将 "Python" 和 "JAVA" 转换为小写字母

```sql
SELECT LCASE('Python'), LCASE('JAVA');
+------------------+-----------------+
| LCASE('Python')  | LCASE('JAVA')   |
+------------------+-----------------+
| python           | java            |
+------------------+-----------------+
```
