### 反转函数

#### 概述
反转函数用于将输入的字符串或数组的元素顺序颠倒。该函数可以处理不同类型的数据，非常灵活且实用。

#### 语法
```
reverse(str)
reverse(array)
```

#### 参数
- `str` (字符串类型)：需要反转的字符串。
- `array` (数组类型)：需要反转的数组。

#### 返回值
- 返回与输入参数相同类型的数据，但元素顺序相反。

#### 使用示例

1. 反转字符串：
```sql
SELECT reverse("hello");
```
输出：
```
olleh
```

2. 反转字符串（包含特殊字符和空格）：
```sql
SELECT reverse("a b c!@#");
```
结果：
```
#@! c b a
```

3. 反转数组：
```sql
SELECT reverse(array(1, 2, 3));
```
结果：
```
[3, 2, 1]
```

4. 反转数组（包含负数和零）：
```sql
SELECT reverse(array(-1, 0, 2, -3));
```
结果：
```
[-3, 2, 0, -1]
```

5. 反转字符串并结合其他函数进行操作：
```sql
SELECT upper(reverse("hello"));
```
结果：
```
OLLEH
```

#### 注意事项
- 反转函数不会修改原始数据，而是返回一个新的、元素顺序相反的数据副本。
- 当处理大量数据时，反转操作可能会消耗较多的计算资源和时间。

通过以上示例和说明，您可以更轻松地使用反转函数处理各种数据。请根据实际需求进行调整和应用。
