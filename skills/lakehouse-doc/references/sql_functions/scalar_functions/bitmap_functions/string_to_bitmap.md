### STRING_TO_BITMAP 函数

```
string_to_bitmap(str)
```

#### 功能描述

`STRING_TO_BITMAP` 函数用于将逗号分隔的字符串转换为 bitmap。

#### 参数说明

* `str`：`STRING` 类型，逗号分隔的整数字符串。

#### 返回类型

* 返回 `BITMAP` 类型。

#### 注意事项

* 如果字符串包含负数或格式不正确，返回 `NULL`。

#### 使用示例

1. 将字符串转换为 bitmap

```sql
SELECT string_to_bitmap('1,2,2');
+----------------------------+
| string_to_bitmap('1,2,2')  |
+----------------------------+
| [1,2]                      |
+----------------------------+
```

2. 转换包含不连续整数的字符串

```sql
SELECT string_to_bitmap('1,3,5');
+----------------------------+
| string_to_bitmap('1,3,5')  |
+----------------------------+
| [1,3,5]                    |
+----------------------------+
```

3. 包含负数时返回 NULL

```sql
SELECT string_to_bitmap('1,2,-1');
+------------------------------+
| string_to_bitmap('1,2,-1')   |
+------------------------------+
| NULL                         |
+------------------------------+
```
