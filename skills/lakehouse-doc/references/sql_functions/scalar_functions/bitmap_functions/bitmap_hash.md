### BITMAP_HASH 函数

```
bitmap_hash(expr)
```

#### 功能描述

`BITMAP_HASH` 函数用于对输入的字符串进行哈希运算，返回包含哈希值的 bitmap。

#### 参数说明

* `expr`：`STRING` 类型的表达式。

#### 返回类型

* 返回 `BITMAP` 类型，包含一个整数哈希值。

#### 使用示例

1. 对字符串进行哈希

```sql
SELECT bitmap_hash('hello');
+-----------------------+
| bitmap_hash('hello')  |
+-----------------------+
| [1321743225]          |
+-----------------------+
```

2. 对另一个字符串进行哈希

```sql
SELECT bitmap_hash('world');
+-----------------------+
| bitmap_hash('world')  |
+-----------------------+
| [2453398188]          |
+-----------------------+
```

3. NULL 值处理

```sql
SELECT bitmap_hash(null);
+-------------------+
| bitmap_hash(null) |
+-------------------+
| NULL              |
+-------------------+
```
