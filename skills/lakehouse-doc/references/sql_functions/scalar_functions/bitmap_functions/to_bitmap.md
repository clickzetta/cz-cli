### TO_BITMAP 函数

```
to_bitmap(expr)
```

#### 功能描述

`TO_BITMAP` 函数用于将整数或字符串转换为包含单个元素的 bitmap。

#### 参数说明

* `expr`：整数类型或 `STRING` 类型的表达式。

#### 返回类型

* 返回 `BITMAP` 类型，包含一个元素。

#### 注意事项

* 如果输入为 `NULL` 或负数，返回 `NULL`。

#### 使用示例

1. 将整数转换为 bitmap

```sql
SELECT to_bitmap(0);
+--------------+
| to_bitmap(0) |
+--------------+
| [0]          |
+--------------+
```

2. 将大整数转换为 bitmap

```sql
SELECT to_bitmap(1234567);
+--------------------+
| to_bitmap(1234567) |
+--------------------+
| [1234567]          |
+--------------------+
```

3. 将字符串转换为 bitmap

```sql
SELECT to_bitmap('1234567');
+----------------------+
| to_bitmap('1234567') |
+----------------------+
| [1234567]            |
+----------------------+
```

4. 负数返回 NULL

```sql
SELECT to_bitmap(-1);
+---------------+
| to_bitmap(-1) |
+---------------+
| NULL          |
+---------------+
```

5. NULL 值返回 NULL

```sql
SELECT to_bitmap(null);
+-----------------+
| to_bitmap(null) |
+-----------------+
| NULL            |
+-----------------+
```
