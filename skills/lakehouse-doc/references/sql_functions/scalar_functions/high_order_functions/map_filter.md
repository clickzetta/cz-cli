### MAP_FILTER 函数

#### 功能描述
MAP_FILTER 函数用于根据给定的 lambda 表达式筛选 map 类型的元素。该函数接收两个参数：第一个参数为 map 类型的数据，第二个参数为一个二参形式的 lambda 表达式，分别对应 map 中的键（k）和值（v）。lambda 表达式需要返回一个布尔值，用于判断当前键值对是否应该保留在结果中。

#### 参数说明
- map: 输入的 map<K, V> 类型数据。
- (k, v) -> expr: 一个二参形式的 lambda 表达式，k 和 v 分别代表 map 中的键和值。expr 需要返回一个布尔值，表示当前键值对是否应该被保留。

#### 返回值
返回一个新的 map<K, V> 类型数据，其中仅包含满足 lambda 表达式条件的键值对。
