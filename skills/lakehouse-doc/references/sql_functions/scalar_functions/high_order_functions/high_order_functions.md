### 高阶函数
高阶函数是那些接受一个或多个 lambda 函数作为参数的函数。在 SQL 语言中，lambda 函数通常用于处理数组、映射等复杂数据类型，以及在子查询中进行计算。

#### Lambda 函数语法
Lambda 函数的语法格式为 `arg -> expr` 或者 `(arg1, arg2) -> expr`：
1. `->` 左侧为参数列表，可以表示数组中的元素，也可以表示映射中的键值对。参数绑定的具体含义取决于高阶函数的语义。
2. `->` 右侧为 lambda 函数的计算体，即对形参的计算。在表达式 `expr` 中，可以引用外部查询或子查询中的列名。

#### 使用示例
1. 数组元素处理：
``` sql
SELECT transform(array(1, 2, 3), x -> x + 1);
-- 结果：[2, 3, 4]
```
上述示例中，我们对数组 `[1, 2, 3]` 中的每个元素 `x` 应用 lambda 函数 `x -> x + 1`，得到新数组 `[2, 3, 4]`。

2. 映射元素处理：
``` sql
SELECT transform(map('a', 1, 'b', 2), (k, v) -> k || ' : ' || v);
-- 结果：{'a' : 'a : 1', 'b' : 'b : 2'}
```
在这个例子中，我们对映射 `{'a' : 1, 'b' : 2}` 中的每个键值对 `(k, v)` 应用 lambda 函数 `(k, v) -> k || ' : ' || v`，得到新的映射 `{'a' : 'a : 1', 'b' : 'b : 2'}`。

3. 捕获外部列：
``` sql
SELECT transform(array(1, 2, 3), x -> x + a) FROM (SELECT 100 AS a) AS sub;
-- 结果：[101, 102, 103]
```
在这个例子中，我们从子查询中捕获列 `a` 的值，并将其与数组 `[1, 2, 3]` 中的元素 `x` 相加，得到新数组 `[101, 102, 103]`。

#### 注意事项
1. Lambda 函数的参数列表应与高阶函数期望的参数类型和数量相匹配。
2. 在 lambda 函数中，可以引用外部查询或子查询中的列名，但需确保其可读性和正确性。
3. 高阶函数的具体用法和语义因函数而异，使用前请仔细阅读相关文档。
