### STRUCT 函数

#### 功能描述
`STRUCT` 函数用于创建一个结构化对象，该对象包含指定数量的字段（field），每个字段由其名称和对应的值组成。通过使用 `STRUCT` 函数，可以将多个不同类型的数据值组合成一个单一的数据结构，即结构化对象。

#### 语法
```
STRUCT(v1, v2, ..., vN)
```

#### 参数
- `v1`, `v2`, ..., `vN`: 任意数量的数据值，可以是不同的数据类型。

#### 返回结果
返回一个结构化对象，其中包含 N 个字段，字段名称默认为 `"col1"`, `"col2"`, ..., `"colN"`，字段值对应传入的参数 `v1`, `v2`, ..., `vN`。

#### 使用示例
1.  创建一个包含两个字段的结构化对象，字段名为 `"age"` 和 `"name"`，分别对应年龄和姓名：
```sql
SELECT STRUCT(25 , 'John Doe');
```
返回结果：
```
{"col1":25,"col2":"John Doe"}
```

2.  将不同类型的数据（整数、字符串、布尔值）组合成一个结构化对象：
```sql
SELECT STRUCT(42, 'Hello, world!', TRUE);
```
返回结果：
```
{"col1":42,"col2":"Hello, world!","col3":true}
```
