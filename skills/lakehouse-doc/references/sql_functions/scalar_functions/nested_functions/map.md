### MAP 函数

#### 功能描述
MAP 函数用于创建一个键值对集合，其中每个键（key）对应一个值（value）。该函数可以一次性插入多个键值对，方便用户快速构建一个 Map 类型的数据结构。

#### 语法格式
```
MAP(k1, v1, ..., kN, vN)
```

#### 参数说明
- k1, ..., kN : 表示函数的键参数，类型为 `K`。
- v1, ..., vN : 表示函数的值参数，类型为 `V`。

#### 返回类型
- 返回一个 `map<K, V>` 类型的对象。

#### 使用示例
1. 创建一个包含两个键值对的 Map：
   ```sql
   SELECT MAP("name", "Alice", "age", '25');
   // 返回：{"name":"Alice","age":"25"}
   ```
2. 构建一个包含三个键值对的 Map，其中包含重复的键（但值不同）：
   ```sql
   SELECT MAP("fruit", "apple", "fruit", "banana", "color", "green");
   // 返回：{"fruit":"banana","color":"green"}
   ```
3. 向 Map 中插入不同类型的键值对（字符串和整数）：
   ```sql
   SELECT MAP("width", 100, "height", 200);
   // 返回：{"width":100,"height":200}
   ```
4. 创建一个空的 Map 对象：
   ```sql
   SELECT MAP();
   // 返回：{}
   ```

#### 注意事项
- 确保提供的键和值数量匹配（成对出现），否则会导致函数执行失败。
- 如果存在重复的键，后面的键值对会覆盖前面的键值对。
- MAP 函数可以接受任意类型的键和值，但键必须是可比较的类型，以便在 Map 中进行查找操作。
