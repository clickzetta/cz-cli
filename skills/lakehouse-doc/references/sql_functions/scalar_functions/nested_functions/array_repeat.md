### 数组重复函数：ARRAY_REPEAT

#### 功能描述
`array_repeat` 函数用于创建一个新的数组，该数组由指定数量的相同元素组成。这个函数接受两个参数：`expr` 表示要重复的元素，`count` 表示重复的次数。函数返回一个类型为 `array<T>` 的数组，其中 `T` 是元素的类型。

#### 参数说明
- `expr` (T): 要重复的元素。
- `count` (int): 元素需要重复的次数。

#### 返回类型
- 返回一个类型为 `array<T>` 的数组。

#### 使用示例
1.  重复字符串：
    ```sql
   SELECT array_repeat('hello', 3);
   ```
    结果：
    ```
   ["hello", "hello", "hello"]
   ```
2.  重复数值：
    ```sql
   SELECT array_repeat(42, 4);
   ```
    结果：
    ```
   [42, 42, 42, 42]
   ```
3.  结合其他函数使用：
    ```sql
   SELECT array_repeat(upper('a'), 5);
   ```
    结果：
    ```
   ["A", "A", "A", "A", "A"]
   ```
4.  重复日期时间：
    ```sql
   SELECT array_repeat(timestamp '2023-04-01 12:00:00', 2);
   ```
    结果：
    ```
   ["2023-04-01 12:00:00", "2023-04-01 12:00:00"]
   ```

#### 注意事项
- 确保 `count` 参数是一个非负整数。如果 `count` 为负数，函数将返回一个空数组。
- 当 `expr` 是一个复杂的数据类型（如对象或数组）时，整个结构将被重复，而不是单个元素。

通过使用 `array_repeat` 函数，您可以轻松地创建包含重复元素的数组，从而简化数据处理和转换的逻辑。
