### CONCAT 函数

#### 概述
`CONCAT` 函数用于将多个数组、字符串或二进制数据连接起来，生成一个新的数组、字符串或二进制数据。

#### 语法
```
CONCAT(array1, array2, ..., arrayN)
CONCAT(str1, str2, ..., strN)
CONCAT(binary1, binary2, ..., binaryN)
```

#### 功能描述
1. 数组连接：将 `array1` 至 `arrayN` 中的所有元素合并成一个新的数组。
2. 字符串连接：将 `str1` 至 `strN` 中的所有字符串拼接成一个新的字符串。
3. 二进制数据连接：将 `binary1` 至 `binaryN` 中的所有二进制数据拼接成一个新的二进制数据。

#### 参数说明
- `array1` 至 `arrayN`: `array<T>` 类型，表示要连接的数组。
- `str1` 至 `strN`: `string` 类型，表示要连接的字符串。
- `binary1` 至 `binaryN`: `binary` 类型，表示要连接的二进制数据。

#### 返回值
- 数组连接版本：返回一个新的 `array<T>` 类型数组。
- 字符串连接版本：返回一个新的字符串。
- 二进制数据连接版本：返回一个新的二进制数据。

#### 使用示例
```sql
-- 数组连接示例
SELECT CONCAT(array(1, 2), array(3, 4));
-- 结果：[1, 2, 3, 4]

-- 字符串连接示例
SELECT CONCAT('hello', '-', 'world');
-- 结果：'hello-world'

-- 二进制数据连接示例
SELECT CONCAT(CAST('123' AS BINARY), CAST('456' AS BINARY));
-- 结果：'123456'
```

#### 注意事项
- 当连接空数组或空字符串时，`CONCAT` 函数将返回一个空数组或空字符串。
- 当连接多个二进制数据时，请确保它们的长度相同，否则可能导致无法正确连接。
- 在连接字符串时，如果需要在连接的字符串之间添加分隔符，可以将其作为一个额外的参数传递给 `CONCAT` 函数。

#### 总结
`CONCAT` 函数提供了一种简单且高效的方式来连接数组、字符串和二进制数据。通过使用此函数，您可以轻松地将多个数据组合成一个统一的数据结构，以便进一步处理和分析。
