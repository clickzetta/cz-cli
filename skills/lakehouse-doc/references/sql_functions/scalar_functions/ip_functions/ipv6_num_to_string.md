### IPv6 转换为字符串

#### 功能描述

`ipv6_num_to_string` 函数用于将二进制形式的 IPv6 地址转换为可读的字符串形式。该函数接受一个二进制类型的参数，并返回一个字符串类型的结果。

#### 参数说明

* `expr`: 二进制类型的 IPv6 地址。

#### 返回结果

* 返回一个字符串类型的 IPv6 地址。

#### 使用示例

以下示例展示了如何使用 `ipv6_num_to_string` 函数将二进制形式的 IPv6 地址转换为可读的字符串形式：

将 IPv6 地址 `::ffff:127.0.0.1` 从字符串形式转换为二进制形式，然后再转换回字符串形式：

```sql
SELECT ipv6_num_to_string(ipv6_string_to_num('::ffff:127.0.0.1'));
+------------------+
|       res        |
+------------------+
| ::ffff:127.0.0.1 |
+------------------+
```

#### 注意事项

* 请确保输入的二进制参数是有效的 IPv6 地址。
* 在转换过程中，如果输入的参数不是有效的 IPv6 地址，函数将返回空字符串。
