### IPv4_NUM_TO_STRING 函数

#### 功能描述
`ipv4_num_to_string` 函数用于将 IPv4 地址的数字形式（只考虑低位的 32 位）转换为可读的字符串形式。该函数接收一个 bigint 类型的参数，并返回一个字符串类型的 IPv4 地址。

#### 参数说明
* `expr`: 一个 bigint 类型的参数，代表 IPv4 地址的数字形式。

#### 返回结果
* 返回一个字符串，表示传入的 IPv4 地址的可读形式。

#### 使用示例
以下展示了 `ipv4_num_to_string` 函数的几个使用示例：

```sql
-- 例子 1: 转换一个常见的回环地址
SELECT ipv4_num_to_string(2130706433); -- 返回 '127.0.0.1'

-- 例子 2: 转换一个公网地址
SELECT ipv4_num_to_string(3232235777); -- 返回 '192.168.1.1'

-- 例子 3: 转换一个私有地址
SELECT ipv4_num_to_string(1730150097); -- 返回 '103.31.254.209'

-- 例子 4: 转换一个特殊用途的地址
SELECT ipv4_num_to_string(2130706432); -- 返回 '127.0.0.0'
```

通过上述示例，我们可以看到 `ipv4_num_to_string` 函数能够准确地将数字形式的 IPv4 地址转换为对应的字符串形式，方便用户进行网络相关的查询和操作。
