### MINUTE 函数

```
MINUTE(timestamp)
```

#### 功能描述

MINUTE 函数用于从给定的 timestamp 类型表达式中提取分钟部分。该函数接受一个 timestamp 类型的参数，并返回一个整数，表示该时间戳的分钟部分。

#### 参数说明

* `timestamp`：一个 timestamp 类型的表达式，可以是字符串、timestamp 值等。

#### 返回结果

返回一个整数，表示 timestamp 参数中的分钟部分。

#### 使用示例

1. 从字符串类型的时间中提取分钟部分：

```
SELECT MINUTE('2022-01-01 10:23:15.321');
-- 结果：23
```

2. 从当前时间戳中提取分钟部分：

```
SELECT MINUTE(TIMESTAMP "2022-01-02 03:21:00");
-- 结果：21
```

3. 从字符串类型的时间中提取分钟部分，并与其他时间操作结合使用：

```
SELECT MINUTE('2022-01-01 10:23:15.321') + INTERVAL '1' MINUTE;
-- 结果：24
```

4. 从当前时间戳中提取分钟部分，并将其转换为字符串格式：

```
SELECT CAST(MINUTE(TIMESTAMP "2022-01-02 03:21:00") AS STRING);
-- 结果：21
```

通过以上示例，您可以看到 MINUTE 函数在不同场景下的应用。该函数可以帮助您更方便地处理时间数据，并根据需求提取分钟部分。
