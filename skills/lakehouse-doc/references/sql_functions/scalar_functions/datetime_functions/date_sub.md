DATE\_SUB 函数

功能
----

DATE\_SUB 函数用于计算指定日期加上或减去一定天数后的日期。该函数返回一个新的日期值。如果计算结果超出系统日期范围，则返回 null。

语法
--

```
date_sub(date, int)
```

参数
----

* `date`: 需要进行日期计算的日期，类型为 date。
* `int`: 需要加上或减去的天数，类型为整数。

返回结果
-------

返回一个新的日期值，类型为 date。

示例
----

以下示例展示了如何使用 DATE\_SUB 函数进行日期计算：

1. 计算 2020 年 5 月 31 日 3 天前的日期：

   ```
   SELECT date_sub('2020-05-31', 3);
   ```

   结果：

   ```
   2020-05-28
   ```


2. 计算 2020 年 2 月 29 日前 1 天后的日期。

   ```
   SELECT date_sub('2020-02-29', 1);
   ```

   结果：

   ```
   2020-02-28
   ```



请注意，DATE\_SUB 函数仅适用于具有日期类型的列。如果输入的参数类型不正确，将返回错误。
