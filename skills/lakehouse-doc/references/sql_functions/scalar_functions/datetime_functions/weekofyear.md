### 函数名称
`WEEKOFYEAR`

### 功能描述
`WEEKOFYEAR` 函数用于计算并返回给定日期（`date` 或 `timestamp_ltz` 类型）在当年是第几个星期。需要注意的是，该函数将一周的开始定义为星期一，并且每年的第一周是指包含该年第一个星期四（即有四天或更多天在当年）的星期。

### 参数说明
- `expr`: 输入的日期或时间戳（`date` 或 `timestamp_ltz`）。

### 返回类型
- 返回一个整数，表示给定日期在当年是第几个星期。

### 使用示例
1. 计算某个特定日期是第几个星期：
   ```sql
   SELECT WEEKOFYEAR('2022-03-31'); -- 结果为：13
   ```
2. 对于当前时间戳，计算其所在的星期数：
   ```sql
   SELECT WEEKOFYEAR(CURRENT_TIMESTAMP()); -- 结果将根据当前时间戳计算
   ```

   
### 注意事项
- 该函数对于输入的日期格式有一定的要求，确保输入的日期格式正确无误。
- 当输入的时间戳（`timestamp_ltz`）跨越了当年的第一个星期四时，返回的结果可能会有所不同。
- 在进行日期计算时，需要注意时区的影响，尤其是对于跨越不同时区的日期和时间戳。
