### 函数名称

QUARTER

### 功能描述

DATE\_QUARTER 函数用于获取指定日期值的季度信息。季度是按照一年分为四个时间段来计算的，每个季度包含三个月。该函数将输入的日期值转换为对应的季度，返回一个整数表示季度。

### 语法

```
QUARTER(date)
```

#### 参数说明

* **date** (date): 需要提取季度的日期值。

### 返回结果

返回一个整数，表示输入日期所在的季度。1 表示第一季度（1-3月），2 表示第二季度（4-6月），3 表示第三季度（7-9月），4 表示第四季度（10-12月）。

### 示例

以下是一些使用 QUARTER 函数的示例：

1. 计算当前日期的季度：

```sql
SELECT QUARTER(CURRENT_TIMESTAMP()) as res;
+-----+
| res |
+-----+
| 1   |
+-----+
```
2. 计算指定日期的季度：

```sql
SELECT QUARTER(date'2022-12-31') as res;
+-----+
| res |
+-----+
| 4   |
+-----+
```
### 注意事项

*   输入的日期值必须是有效的日期格式，否则会导致函数执行失败。
*   如果输入的日期值是当前时间或未来的日期，返回的季度信息仍然有效。
