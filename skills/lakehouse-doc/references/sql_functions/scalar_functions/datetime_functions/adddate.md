### ADD_DAYS 函数
#### 功能描述
`ADD_DAYS` 函数用于在给定的日期 `startDate` 上增加 `numDays` 天，返回一个新的日期。如果计算结果超出了系统所支持的日期范围，则返回 `null`。

#### 函数语法
```
add_days(startDate, numDays)
```
#### 参数说明
- `startDate`: 日期类型，表示开始日期。
- `numDays`: 整数类型，表示要增加的天数。

#### 返回结果
返回一个新的日期，类型为 `date`。

#### 使用示例
1. 增加 5 天到指定日期：
```
SELECT add_days('2022-01-01', 5);
```
结果：
```
2022-01-06
```
2. 从指定日期减少 10 天：
```
SELECT add_days('2022-01-15', -10);
```
结果：
```
2022-01-05
```
3. 计算 100 天后的重要会议日期：
```
SELECT add_days('2022-03-01', 100);
```
结果：
```
2022-06-09
```
4. 计算从某项目启动日期起 200 天后的结束日期：
```
SELECT add_days('2022-01-20', 200);
```
结果：
```
2022-08-18
```
#### 注意事项
- 确保 `startDate` 参数为有效的日期格式，否则会导致函数返回 `null`。
- 当增加的天数导致日期超出系统支持范围时，函数将返回 `null`，请注意检查返回值。

通过使用 `ADD_DAYS` 函数，您可以方便地计算未来或过去的日期，从而在处理日期相关问题时提供极大的便利。
