### 获取当前工作区ID函数
```sql
current_workspace_id()
```
#### 功能描述
`current_workspace_id()` 函数用于获取当前会话所在的工作区`ID`。该函数无需传入任何参数,将返回当前会话所关联的工作区的唯一标识符（整数值）。

#### 使用场景
在多工作区的环境中,需要根据当前用户所在的工作区执行特定操作时,可以使用此函数获取当前工作区的`ID`。

#### 返回结果
返回值类型为整数（`INT`）,表示当前会话所在的工作区`ID`。

#### 使用示例
以下示例展示了如何使用 `current_workspace_id()` 函数获取当前工作区`ID`,并根据不同的工作区执行不同的操作。

示例 1：查询当前工作区ID
```sql
SELECT current_workspace_id();
```
执行结果：
```
123
```
