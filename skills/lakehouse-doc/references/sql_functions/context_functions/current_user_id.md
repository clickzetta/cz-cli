### CURRENT_USER_ID 函数

#### 功能描述
`CURRENT_USER_ID` 函数用于获取当前数据库会话用户的 `ID`。这个函数可以帮助您快速了解当前操作用户的身份，从而进行权限控制或其他相关操作。

#### 使用方法
`CURRENT_USER_ID` 函数不需要任何参数。您只需在 `SQL` 查询中调用此函数，它就会返回当前会话用户的 `ID`。

#### 返回结果
返回值类型为整数（`INT`），表示当前会话用户的 `ID`。

#### 使用示例

1. 查询当前用户的ID并显示结果：
```sql
SELECT CURRENT_USER_ID();
```
执行上述查询后，您将看到类似于以下的输出，它表示当前会话用户的 `ID`：
```
123
```
