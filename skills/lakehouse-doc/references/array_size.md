### 函数名称：SIZE

#### 功能描述
1. `SIZE(array)` 函数用于计算并返回数组（array）中元素的数量。
2. `SIZE(map)` 函数用于计算并返回映射（map）中键值对的数量。

#### 语法
```
SIZE(array)
SIZE(map)
```

#### 参数说明
- `array`: 数组类型 `array<T>`，表示需要计算元素个数的数组。
- `map`: 映射类型 `map<K, V>`，表示需要计算键值对个数的映射。

#### 返回类型
整数（int）

#### 使用示例
1. 计算数组中元素的个数：
```
SELECT SIZE(array(1, 2, 3)); -- 结果为：3
```
2. 计算映射中键值对的个数：
```
SELECT SIZE(map("a", 1, "b", 2)); -- 结果为：2
```

#### 注意事项
- 当传入的参数不是数组或映射时，函数将返回错误。
- 如果数组或映射为空，函数将返回0。
