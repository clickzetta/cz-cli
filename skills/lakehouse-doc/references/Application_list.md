# 申请列表

通过申请记录功能，您可以查看所有自己申请的工单，包括申请内容、申请状态等，还可以进行工单撤销或复制操作。

## 前提条件

已申请资源权限，相关操作说明请参见**权限申请**。

## 操作步骤

点击左侧菜单栏的**审批 > 申请**，进入**申请记录**页面。

![](.topwrite/assets/image_1736245885982.png)

可执行以下操作：
![](.topwrite/assets/image_1736245926683.png)

* **查看工单详情**：单击列表中的某个**工单ID**，在弹出的侧拉窗口中，可以查看该工单的详情，包括申请详情、申请资源、审批情况等。

* **撤回**：对审批中的工单，可执行撤回操作。

  * 单击列表中某条信息的**操作**列或工单详情页面的**撤回**按钮，在弹出的确认撤销对话框中，单击**确定**按钮，可以撤销对应资源的申请。撤销时会撤回工单内的所有在途资源申请，工单内已审批通过的资源则不受影响。

* **复制**：单击列表中某条信息的**操作**列或工单详情页面的**复制**按钮，可将当前审批的信息全部复制，可按需修改相关信息后提交申请。
