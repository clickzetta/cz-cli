Execute ClickZetta Lakehouse CLI commands via cz-tool. Use this tool for database operations including SQL queries, table management, schema operations, task scheduling, and profile management.

Commands are passed as a single string and executed by the built-in cz-cli engine. Output is always JSON for machine parsing.

Common commands:
- sql "SELECT * FROM table LIMIT 10" — execute SQL
- table list — list tables
- table describe <name> — describe table schema
- schema list — list schemas
- profile list — list connection profiles
- task list — list scheduled tasks
- runs list — list task run instances
- status — check connection status

The --output json flag is automatically appended. Use --profile <name> to target a specific connection profile.
